# DemonstrativoⓍ

Acesse:

**Controladoria → Desbloqueio**

1. Informe os filtros desejados;
2. Clique em **Gerar** para emitir o demonstrativo.

**Filtros**

Nesta tela são definidos os parâmetros utilizados para gerar o relatório.

Preencha os principais campos, como:

- Data Inicial;
- Data Final;
- Tipo;
- Opções.

Esses filtros determinam quais informações serão consideradas na geração do DRE.

---

**Finalizando**

Após informar os filtros desejados, clique em **Gerar** para emitir o Demonstrativo dos Resultados do Exercício.
