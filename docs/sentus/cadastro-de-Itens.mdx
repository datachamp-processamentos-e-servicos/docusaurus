## Cadastro de Itens

## Como cadastrar um novo item

Para realizar o cadastro de um novo item no sistema, siga o passo a passo abaixo:

1. Acesse o menu **Cadastros**;
2. Clique em **Itens**;
3. Em seguida, clique novamente em **Itens**;
4. No canto superior direito da tela, clique em **+ Novo**.

Ao clicar em **+ Novo**, será aberta a tela de cadastro de item.

O sistema irá sugerir automaticamente, no primeiro campo, a última referência utilizada.

## Preenchimento das informações

A partir deste ponto, você deve preencher os dados do item conforme necessário.

*Todos os campos sinalizados com (*) são de preenchimento obrigatório.*

## Grupo Tributário

O último campo a ser informado é o **Grupo Tributário**.

*Importante: É com base nesse campo que o sistema realiza a tributação das saídas do item.*

Certifique-se de selecionar corretamente o grupo tributário, pois ele impacta diretamente nos cálculos fiscais das operações.
