## Cadastrar Nova Guia de Arrecadação

Acesse:

**Tributos → Guias de Arrecadação**

1. Clique em **+ Novo** para cadastrar uma guia de arrecadação ou clique em um registro existente para editá-la;
2. Preencha as informações da guia;
3. Clique em **Salvar** para concluir o cadastro.

## Informações da Guia

Nesta seção são informados os dados utilizados para emissão e controle da guia de arrecadação.

Preencha os principais campos, como:

- Emissão;
- Vencimento;
- Pagamento;
- Valor;
- Emitente;
- Código DA;
- Estado;
- Número DA;
- Autenticação;
- Código da Obrigação;
- Código da Receita;
- Código de Apuração do Recolhimento;
- Descrição Complementar;
- Origem;
- Origem ID;
- Observação.

Essas informações permitem identificar a guia, controlar seu pagamento e registrar os dados necessários para o recolhimento do tributo.

---

**Finalizando**

Após revisar todas as informações, clique em **Salvar** para concluir o cadastro da guia de arrecadação.
