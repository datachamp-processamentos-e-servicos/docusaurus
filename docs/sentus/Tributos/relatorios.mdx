## Gerar Relatórios

Acesse:

**Jurídico → Relatórios**

1. Selecione o relatório desejado na lista.
2. Preencha os parâmetros solicitados.
3. Clique em **Visualizar** para gerar o relatório.
4. Utilize **PDF** ou **Outros Formatos** para exportar o resultado.

## Informações dos Relatórios

A tela apresenta todos os relatórios disponíveis para execução.

As principais informações exibidas são:

- **Código:** identificação do relatório.
- **Título:** nome do relatório.
- **Módulo:** módulo ao qual pertence.
- **Orientação:** orientação padrão de impressão.
- **Ações:** executa o relatório selecionado.

Após selecionar um relatório, o sistema solicitará os parâmetros necessários para sua geração.

## Exportação

Após gerar o relatório, é possível exportá-lo nos seguintes formatos:

- PDF
- Excel
- CSV
- TXT

Dependendo do formato selecionado, também é possível definir o tipo de exportação:

- **0 - Não se Aplica**
- **1 - Entre Linhas**
- **8 - Um PDF por Agrupamento**

**Finalizando**

Após gerar ou exportar o relatório, o arquivo estará disponível para visualização ou download conforme o formato escolhido.
