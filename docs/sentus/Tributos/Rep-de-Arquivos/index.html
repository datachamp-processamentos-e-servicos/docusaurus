Repositório de Arquivos

Visão Geral​

A tela de Repositório de Arquivos é utilizada para gerar, armazenar, consultar e compartilhar arquivos fiscais e obrigações acessórias.

Por meio desta funcionalidade, é possível gerar arquivos SPED, EFD, DIRF, REINF e demais documentos fiscais exigidos pelos órgãos competentes.

Observações Importantes​

  • Todos os campos marcados com (*) são obrigatórios.
  • Alguns campos são restritos ao administrador do sistema.
  • Verifique o período informado antes de gerar o arquivo.
  • Confirme o tipo de escrituração (Original ou Retificadora) antes da geração.
  • Revise os destinatários e anexos antes de realizar o envio por e-mail.

Principais Funcionalidades​

Novo Cadastro​

Clique em + Novo para gerar um novo arquivo fiscal.

Preencha os campos obrigatórios e clique em Salvar para concluir o processo.

Informações do Arquivo​

Data Inicial​

Define a Data inicial utilizada para geração das informações.

Tipo de Arquivo​

Determina qual arquivo fiscal será gerado.

Opções disponíveis:

  • EFD Fiscal
  • EFD Contribuições
  • EFD REINF
  • DIRF
  • EFD Fiscal ( Contábil)
  • EFD Fiscal ( Bloco K)
  • EFD Fiscal ( Bloco H)
  • ECD - Bloco I
  • SISDEVIN
  • PER/DCOMP

Finalidade​

Define o tipo de escrituração.

Opções:

  • Original
  • Retificadora

Inventário​

Define quando o inventário será considerado na geração do arquivo.

Bloco K​

Permite configurar informações relacionadas ao estoque escriturado.

Produto Acabado​

Define o tratamento dos produtos acabados no inventário.

Observação​

Campo preenchido automaticamente conforme as configurações selecionadas.

Menu​

Exportar​

Permite exportar os registros exibidospara uma planilha Excel.

Pesquisa Avançada​

Utilize a Pesquisa Avançada para localizar arquivos já gerados.

Filtros Disponíveis​

Somente Emitente Selecionado​

Pesquisa apenas os emitentes previamente selecionados.

Tipo de Arquivo​

Permite localizar registros através do tipo de arquivo gerado.

Filtro por dia​

Permite pesquisar registros por período.

O sistema realiza a consulta através da data de emissão do arquivo.

Mais Ações​

Download​

Realiza o download do arquivo gerado.

Editar​

Permite alterar as configurações do registro.

Visão Geral​

Exibe um resumo das informações fiscais geradas.

São apresentadas informações como:

  • CFOP

  • Quantidade de Itens

  • Valores Totais

  • Base de Cálculo

  • ICMS

  • IPI

  • Diferenças Fiscais

  • Enviar E-mail​

Permite encaminhar o arquivo para um ou mais destinatários.

Nome​

Descrição utilizada para identificar o destinatário.

E-mail​

Endereço eletrônico qe receberá o arquivo.

Anexos​

Define quais arquivos serão enviados.

Opções disponíveis:

  • Sem Anexos
  • PDF
  • XML
  • Boleto
  • PDF + XML
  • PDF + XML + Boleto
  • Repositório Arquivo

Após preencher os dados, clique em Add para incluir o destinatário na lista de envio.

Dúvidas Frequentes​

Como gerar um novo arquivo fiscal ?​

Clique em + Novo, selecione o tipo de arquivo desejado, informe o período e conclua o cadastro.

Como localizar um arquivo já gerado?​

Utilize a Pesquisa Avançada e filtre pelo tipo de arquivo ou período.

Como baixar um arquivo?​

Selecione o registro desejado e utilize a opção Download.

Como visualizar os dados fiscais do arquivo?​

Utilize a opção Visão Geral disponível em Mais Ações.

Como enviar um arquivo por e-mail?​

Acesse Mais Ações > Enviar E-mail, informe os destinatários e selecione os anexos desejados.

Quais arquivos fiscais podem ser gerados?​

O sistema permite gerar arquivos como EFD Fiscal, EFD Contribuições, REINF, DIRF, ECD, PER/DCOMP e outros modelos disponíveis.
