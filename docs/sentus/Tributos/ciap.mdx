## Cadastrar Controle de Crédito do Ativo Permanente

Acesse:

**Tributos → Imobilizado → CIAP - Controle de Crédito do Ativo Permanente**

1. Clique em **+ Novo** para cadastrar um controle de crédito ou clique em um registro existente para editá-lo;
2. Preencha as informações do período;
3. Clique em **Calcular** para realizar o cálculo dos valores, quando necessário;
4. Clique em **Salvar** para concluir o cadastro.

## Informações do Controle

Nesta seção são informados os dados utilizados para o cálculo e controle dos créditos do ativo permanente.

Preencha os principais campos, como:

- Data Inicial;
- Data Final;
- Valor Tributado;
- Valor Total;
- Índice;
- ICMS Inicial;
- ICMS Somatório;
- ICMS Apropriado;
- ICMS Outros;
- Código do Benefício Fiscal;
- Observação.

Essas informações são utilizados para controlar a apropriação dos créditos de ICMS referentes aos bens do ativo permanente.

---

**Finalizando**

Após revisar todas as informações e realizar o cálculo, quando necessário, clique em **Salvar** para concluir o cadastro.
