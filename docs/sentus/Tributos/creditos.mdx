## Cadastro de Créditos

Acesse:

**Tributos → Lista de Créditos**

1. Clique em **+ Novo** para cadastrar um crédito ou selecione um registro existente para editá-lo;
2. Informe os dados do crédito;
3. Clique em **Salvar** para concluir o cadastro.

## Informações do Crédito

Nesta seção são informados os dados utilizados para controle e apuração dos créditos fiscais.

Preencha os principais campos, como:

- Período;
- Registro;
- Origem;
- Código;
- CNPJ;
- Data Base;
- Valor Apurado;
- Crédito Extemporâneo Apurado;
- Total do Crédito;
- Desconto Anterior;
- Ressarcimento Anterior;
- Compensação Anterior;
- Crédito Disponível;
- Valor Descontado;
- Pedido de Ressarcimento;
- Declaração de Compensação;
- Valor Transferido;
- Valor Outros;
- Saldo Credor Futuro.

Essas informações são utilizadas para controlar os créditos disponíveis e suas movimentações fiscais.

---

**Finalizando**

Após revisar as informações, clique em **Salvar** para concluir o cadastro do crédito.
