## Cadastrar Novo Certificado

Acesse:

**Administrativo → Certificados**

1. Clique em **+ Novo** para cadastrar um certificado ou clique em um registro existente para editá-lo;
2. Preencha as informações do certificado;
3. Faça o upload do arquivo do certificado A1 e informe a senha;
4. Clique em **Salvar** para concluir o cadastro.

## Informações do Certificado

Nesta tela são cadastrados os certificados digitais utilizados pelo sistema.

Preencha os principais campos, como:

- Nome;
- Válido Até;
- Observação;
- Informações Adicionais.

Essas informações identificam o certificados e permitem acompanhar sua validade.

---

## Certificado A1

Nesta seção é relizado o envio do arquivo do certificado digital.

Preencha os principais campos, como:

- Arquivo do Certificado;
- Senha;
- Repetir a Senha.

Também é posível utilizar as opções:

- **Upload** para enviar o certificado;
- **Testar** para validar o certificado;
- **Download** para baixar o certificado cadastrado.

---

**Finalizando**

Após revisar as informações, clique em **Salvar** para concluir o cadastro do certificado.
