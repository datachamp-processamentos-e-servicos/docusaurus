## Importar Arquivos Individuais

Acesse:

**Administrativo → Configuração → Csv**

1. Informe o **Separador**, quando necessário.
2. Defina a **Data de Emissão**.
3. Cole ou informe os dados no campo **Arquivos Individuais**.
4. Clique em **Opções** para executar a importação.

## Informações da Importação

Nesta tela é possível importar arquivos individuais para processamento pelo sistema.

Os principais campos são:

- **Separador:** caractere utilizado para separar os dados do arquivo.
- **Emissão:** data considerada para a importação.
- **Arquivos Individuais:** área para informar o conteúdo dos arquivos que serão processados.

## Opções de Importação

O menu **Opções** reúne as rotinas de importação e atualização de dados do sistema.

As opções disponíveis são:

- Importar Itens
- Atualizar Itens
- Importar Clientes
- Importar Fornecedores
- Importar Representantes
- Importar Transportadores
- Importar Contatos
- Atualizar Participantes
- Atualizar Situação de Participantes
- Importar CEST
- Importar Títulos
- Importar Código de Barras (Gera SQL)
- Importar Estoques - Entrada
- Importar Estoques - Saída
- Importar NCMs (Reforma Tributária)
- Parâmetros (Itens ou Participantes)

**Finalizando**

Após preencher as informações, utilize o menu **Opções** para iniciar o processamento dos arquivos.
