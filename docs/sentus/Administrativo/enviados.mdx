# EnviadosⓍ

**E-mails Enviados**

Acesse:

**Administrativo → E-mails → Enviados**

Nesta tela é possível consultar o histórico de e-mails enviados pelo sistema.

As principais informações exibidas são:

- **Data:** data e hora do envio.
- **Assunto:** assunto do e-mail.
- **Destinatários:** e-mails que receberam a mensagem.
- **Anexos:** arquivos enviados.
- **Situação:** status do envio.
- **Observações:** informações sobre o processamento.
- **Ações:** permite reenviar o e-mail.
