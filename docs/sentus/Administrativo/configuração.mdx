## Configurar E-mail

Acesse:

**Administrativo → E-mails → Configuração**

1. Clique em **+ Novo** para cadastrar uma configuração ou clique em um registro existente para editá-la;
2. Preencha as informações da configuração;
3. Clique em **Salvar** para concluir o cadastro.

## Informações da Configuração

Nesta tela são definidas as configurações utilizadas para o envio de e-mails pelo sistema.

Preencha os principais campos, como:

- Código;
- Descrição;
- Senha;
- Configurações (Somente Suporte).

:::danger[(Importante)]
As configurações disponíveis na seção **Configurações (Somente Suporte)** devem ser alteradas apenas pela equipe de suporte.
:::

---

**Finalizando**

Após revisar as informações, clique em **Salvar** para concluir o cadastro da configuração de e-mail.
