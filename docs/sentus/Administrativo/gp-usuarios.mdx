## Cadastrar Novo Grupo de Usuários

Acesse:

**Administrativos → Grupos de Usuários**

1. Clique em **+ Novo** para cadastrar um grupo de usuários ou clique em um registro existente para editá-lo.
2. Preencha as informações do grupo;
3. Clique em **Salvar** para concluir o cadastro.

## Informações do Grupo de Usuários

Nesta tela são cadastrados os grupos utilizados para organizar usuários e definir permissões de acesso.

Preencha os principais campos, como:

- ID;
- Nome;
- Observação.

Essas informações permitem identificar e organizar os grupos de usuários do sistema.

---

**Finalizando**

Após revisar as informações, clique em **Salvar** para concluir o cadastro do grupo de usuários.
