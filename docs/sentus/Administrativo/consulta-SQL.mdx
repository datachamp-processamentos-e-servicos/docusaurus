## Executar Relatório SQL

Acesse:

**Administrativo → Relatório SQL**

1. Informe a consulta SQL no campo disponível.
2. Clique em **Executar** para visualizar o resultado.
3. Se desejar, clique em **Exportar** para gerar um arquivo com os dados retornados.

## Informações do Relatório SQL

Esta tela permite executar consultas SQL diretamente no banco de dados para consulta de informações.

Os principais recursos são:

- **Consulta SQL:** área destinada à inserção do comando SQL.
- **Executar:** executa a consulta informada.
- **Exportar:** exporta o resultado da consulta para um arquivo.

## Exportando o Resultado

Após executar a consulta, clique em **Exportar** para gerar um arquivo com os dados retornados.

As opções disponíveis são:

- **Formato de Exportação:**
  - PDF
  - Excel
  - CSV
  - TXT

- **Orientação da Página** (disponível para PDF):
  - Automático
  - Retrato
  - Paisagem

Após selecionar as opções desejadas, clique em **Exportar** para gerar o arquivo.

**Finalizando**

Após executar a consulta, analise os dados retornados ou utilize a opção **Exportar** para salvar o resultado.
