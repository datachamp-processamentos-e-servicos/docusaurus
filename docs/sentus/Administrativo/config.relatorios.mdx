## Cadastrar Novo Relatório

Acesse:

**Administrativo → Configuração → Relatórios**

1. Clique em **+ Novo** para cadastrar um relatório ou selecione um existente para edição.
2. Preencha as informações necessárias.
3. Faça o upload do arquivo do relatório, se necessário.
4. Configure os parâmetros e colunas do relatório.
5. Clique em **Salvar**.

## Informações do Relatório

Os principais campos disponíveis são:

- **Título:** nome do relatório.
- **Módulo:** módulo em que o relatório ficará disponível.
- **Resource:** recurso utilizado pelo relatório.
- **Orientação:** define a orientação da impressão.
- **Tamanho da Fonte:** tamanho da fonte utilizada na impressão.
- **Separador:** caractere utilizado na exportação dos dados.
- **Leiaute:** modelo de apresentação do relatório.
- **Privado:** define se o relatório será visível apenas ao usuário.
- **Descrição:** informações sobre a finalidade do relatório.
- **Informações Adicionais:** observações complementares.
- **Query:** consulta SQL utilizada para gerar os dados.
- **Relatório:** permite realizar o upload do arquivo do relatório.

## Parâmetros

Nesta seção são definidos os parâmetros que serão solicitados na execução do relatório.

Os principais campos são:

- **Posição:** ordem de exibição.
- **Nome Campo:** descrição apresentada ao usuário.
- **Parâmetro:** nome utilizado na consulta.
- **Tipo:** tipo de dado do parâmetro.
- **SQL ou Enum:** consulta ou lista de opções.
- **Default:** valor padrão.
- **Opcional:** define se o preenchimento é obrigatório.

Após configurar os campos, clique em **Add** para adicioná-los ao relatório.

## Colunas

Permite configurar as colunas exibidas no resultado do relatório.

Os principais campos são:

- **Posição:** ordem da coluna.
- **Gráfico:** indica se a coluna será utilizada em gráficos.
- **Agrupar:** define agrupamento dos resultados.
- **Visível:** controla a exibição da coluna.
- **Tamanho:** largura da coluna.
- **Fórmula:** expressão utilizada para cálculo ou apresentação.

Clique em **Add** para incluir a configuração na lista.

**Finalizando**

Após concluir todas as configurações, clique em **Salvar** para disponibilizar o relatório no sistema.
