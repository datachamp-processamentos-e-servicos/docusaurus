## Importar Arquivos

Acesse:

**Administrativo → Importador**

1. Selecione o **Tipo de Arquivo**;
2. Informe o **Separador**, quando necessário;
3. Defina a **Data de Apuração** e o **Depósito**;
4. Selecione o arquivo para importação;
5. Clique em **Upload** para iniciar o processo.

## Informações da Importação

Nesta tela são definidos os parâmetros utilizados para importar arquivos para o sistema.

Preencha os principais campos, como:

- Tipo de Arquivo;
- Separador;
- Data de Apuração;
- Depósito.

Na seção **Upload Arquivo**, selecione o arquivo que será importado e utilize o botão **Upload** para enviar o arquivo ao sistema.

---

**Finalizando**

Após configurar as informações e realizar o upload do arquivo, aguarde a conclusão do processo de importação.
