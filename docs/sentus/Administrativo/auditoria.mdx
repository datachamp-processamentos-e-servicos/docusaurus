## Consultar Auditoria

Acesse:

**Administrativo → Auxiliares → Auditoria**

1. Selecione o registro que deseja consultar;
2. Visualize as informações da auditoria;
3. Utilize os dados para acompanhar alterações realizadas no sistema.

## Informações da Auditoria

Nesta tela são exibidas as informações registradas durante o processo de auditoria.

Os principais campos são:

- Emissão;
- Data do Envio;
- Origem;
- Origem ID;
- Posição;
- Campo;
- Valor;
- Observação.

Essas informações permitem identificar quando uma alteração foi realizada, sua origem e os dados envolvidos.

---

**Finalizando**

Após consultar as informações, utilize os registros apresentados para acompanhar o histórico de alterações do sistema.
