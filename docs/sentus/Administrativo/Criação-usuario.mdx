## Cadastrar Novo Usuário

Acesse:

**Administrativo → Usuários**

1. Clique em **+ Novo** para cadastrar um usuário ou clique em um registo existente para editá-lo;
2. Preencha as informações do usuário;
3. Defina a senha de acesso;
4. Clique em **Salvar** para concluir o cadastro.

## Informações do Usuário

Nesta tela são configurados os dados de acesso do usuário ao sistema.

Preencha os principais campos, como:

- Nome;
- Situação;
- Grupo;
- E-mail;
- Papel;
- Tempo de Sessão (Minutos);
- Configuração.

Essas informações definem o perfil e as permissões de acesso do usuário.

---

## Usuário e Senha

Nesta seção são informadas as credenciais de acesso do usuário.

Preencha os principais campos, como:

- Usuário;
- Senha;
- Repetir a Senha.

:::danger[(ATENÇÃO)]
 **Importante:** A senha criada nesta tela é previsória e deve ser alterada pelo usuário no primeiro acesso. Ela expira em **3 dias**.
:::

---

** Finalizando**

Após revisar as informações, clique em **Salvar* para concluir o cadastro do usuário.
