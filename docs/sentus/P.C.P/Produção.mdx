# Produção 

## Visão Geral

A tela de Produção 2.0 é utilizada para controlar o planejamento e execução dos processos produtivos da empresa.

Por meio desta funcionalidade, é possível criar produções, realizar programações, registrar apontamentos de produção e acompanhar a situação de cada processo produtivo.

---

## Observações Importantes

- Todos os campos marcados com (*) são obrigatórios.
- A programação deve ser realizada antes dos apontamentos.
- Os apontamentos registram a execução real da produção.
- Revise as quantidades informadas antes de confirmar os registros.
- Mantenha a situação da produção atualizada para facilitar o acompanhamento do processo produtivo.

---

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para criar uma nova produção.

Preencha os campos obrigatórios marcados com (*) e clique em **Avançar** para continuar o processo.

### Editar

Permite alterar as informações de uma produção já cadastrada.

Caso exista um campo de CPF/CNPJ no cadastro, não altere essa informação após a criação do registro, pois o sistema poderá considerar a alteração como um novo cadastro.

### Dados Gerais

Nesta etapa são informadas as informações iniciais da produção.

#### Emissão

Data de criação da produção.

#### Participante

Empresa ou participante responsável pela produção.

#### Origem

Define a origem do processo produtivo.

#### Observação

Campo destinado a informações complementares relacionadas à produção.

### Programação

A programação é utilizada para planejar os itens que serão produzidos.

Clique em ** Iniciar Programação** para adicionar uma nova programação.

Os principais campos disponíveis são:

- Data
- Referência
- Item
- Quantidade
- Linha
- Observação

Após informar os dados, clique em **Confirmar** para adicionar a programação.

Clique em **Avançar** para seguir para a etapa de apontamentos.

### Apontamentos

Os apontamentos são utilizados para registrar a execução da produção.

Clique em **Iniciar Apontamentos** para incluir um novo apontamento.

Os principais campos disponíveis são:

- Data
- Depósito
- Linha
- Referência
- Item
- Quantidade
- Observação

Após informar os dados, clique em **Confirmar** para registrar o apontamento.

## Menu

### Finalizar Produção

Permite concluir a produção selecionada.

Após a finalização, a produção passa a ser considerada concluída no sistema.

### Exportador

Gera um arquivo PDF contendo os dados da produção.

O relatório será gerado de acordo com os filtros e registros selecionados.

## Exportar

Acesse **Menu > Exportador** para gerar um PDF com as informações da produção.

O arquivo será gerado conforme os registros selecionados na tela.

## Pesquisa Avançada

Utilize a Pesquisa Avançada para localizar produções específicar.

### Filtros Disponíveis

#### Participante

Pesquisa produções através do participante informado.

#### Observação

Pesquisa produções utilizando informações registradas no campo de observação.

#### Situação

Pesquisa produções conforme o status do processo.

#### Filtro por Data

Permite consultar produções por período utilizando a data de emissão.

## Mais Ações

### Imprimir Programação

Gera uma impressão contendo as informações da programação cadastrada.

### Imprimir Apontamento

Gera uma impressão contendo os dados dos apontamentos realizados.

### Situação

Permite consultar e alterar a situação da produção.

Situação Disponíveis:

- Programando
- Programado
- Produzindo
- Finalizado

Após selecionar a situação desejada, clique em **Alterar** para confirmar a atualização.

## Dúvidas Frequentes

### Como criar uma nova produção?

Clique em **+ Novo**, preencha os dados obrigatórios e avance para as etapas de programação e apontamentos.

### Qual a diferença entre Programação e Apontamento?

A Programação é utilizada para planejar a produção.

O apontamento é utilizado para registrar a execução efetiva da produção.

### Como finalizar uma produção?

Acesse **Menu > Finalizar Produção** ou utilize **Mais Ações > Situação** e altere o status para **Finalizado**.

### Como imprimir uma programação?

Selecione o registro desejado e utilize **Mais Ações > Imprimir Programação**.

### Como imprimir um Apontamento?

Selecione o registro desejado e utilize **Mais ações > Imprimir Apontamento**.

### Como localizar uma produção específica?

Utilize a Pesquisa Avançada e filtre por participantes, situação, observação ou período.
