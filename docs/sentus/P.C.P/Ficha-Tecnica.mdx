## Cadastrar uma Ficha Técnica

Acesse:

**P.C.P. → Ficha Técnica**

1. Clique em **+ Novo** para criar uma ficha técnica;
2. Selecione o **Item Produzido**;
3. Informe a **Quantidade** produzida pela ficha;
4. Adicione os insumos utilizados na fabricação;
5. Clique em **Salvar** para concluir o cadastro.

## Informações da Ficha Técnica

A ficha técnica define quais matérias-primas ou componentes serão consumidos durante a produção de um item.

Essas informações são utilizadas pelo sistema para controlar o consumo de materiais, calcular custos de produção e auxiliar no planejamento da fabricação.

Ao cadastrar uma ficha técnica, informe corretamente:

- Item produzido;
- Quantidade produzida pela ficha;
- Insumos utilizados;
- Quantidade consumida de cada insumo;
- Tipo de consumo aplicado ao item.

## Cadastro de Insumos

Nesta seção são adicionados todos os materiais necessários para produzir o item.

Para incluir um insumo:

1. Informe a **Referência** ou pesquise o item consumido;
2. Defina a **Quantidade** utilizada;
3. Caso necessário, informe o percentual de **Perda**;
4. Selecione o **Tipo** de consumo;
5. Clique em **Add** para incluir o insumo na lista.

É possível adicionar quantos insumos forem necessários para compor a ficha técnica.

**Finalizando**

Após revisar os materiais e quantidades informadas, clique em **Salvar** para concluir o cadastro da ficha técnica.
