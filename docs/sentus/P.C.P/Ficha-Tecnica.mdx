# Ficha TécnicaⓍ

## Visão Geral

A tela de Ficha Técnica é utilizada para definir a composição de um produto, informando os insumos necessários para sua produção.

Por meio desta funcionalidade, é possível cadastrar matérias-primas, quantidades consumidas, perdas de produção e demais informações utilizadas no processo produtivo.

---

## Observações Importantes

- Todos os campos marcados com (*) são obrigatórios.
- Revise as quantidades dos insumos antes de salvar a ficha.
- Configure corretamente os percentuais de perda para manter o custo de produção atualizado.
- Utilize as categorias para organizar os insumos da composição.
- O recálculo de custos pode impactar os valores de produção utilizados pelo sistema.

---

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para criar uma nova ficha técnica.

Preencha os campos obrigatórios e clique em **Salvar** para concluir o cadastro.

### Editar

Permite alterar uma ficha técnica já cadastrada.

Caso exista campo de CPF/CNPJ no cadastro, não altere esta informação após a criação do registro, pois o sistema poderá considerar como um novo cadastro.

### Item Produzido

Selecione o produto final que será fabricado.

Este é o item resultante da composição cadastrada na ficha técnica.

### Quantidade Produzida

Informe a quantidade do produto final gerada pela composição cadastrada.

Este campo determina a proporção dos insumos utilizados na produção.

### Insumos

Permite informar os materiais utilizados na fabricação do produto.

Para cada insumo podem ser definidos os seguintes dados:

#### Referência

Código do item utilizado na produção.

#### Item Consumido

Produto ou matéria-prima que será consumida.

#### Quantidade

Quantidade necessária para produzir o item final.

#### (%) Perda

Percentual de perda ou quebra esperado durante a produção.

#### Tipo

Define como o insumo será tratado no processo produtivo.

#### Categoria

Classifica o insumo dentro da composição do produto.

## Menu

### Descrever Variações

Permite informar descrições e agrupamentos para os itens utilizados na ficha técnica.

### Excluir Todos os Insumos

Remove todos os insumos cadastrados na ficha técnica atual.

### Recalcular Custo

Atualiza os custos dos itens cadastrados na ficha técnica com base nos parâmetros selecionados.

### Exportar

Exporta a listagem de fichas técnicas para um arquivo Excel.

## Pesquisa Avançada

Utilize a Pesquisa Avançada para localizar fichas técnicas específicas.

### Filtros Disponíveis

#### Item

Pesquisa pelo item produzido.

#### Tipo

Pesquisa pelo tipo do item.

#### Categoria

Pesquisa pela categoria dos insumos.

#### Insumos

Permite localizar fichas técnicas através dos insumos utilizados.

#### Ordenação

Define a forma de organização dos resultados exibidos. 

## Mais Ações

### Adicionar

Após preencher os dados do insumo, clique em **Add** para incluí-lo na composição da ficha técnica.


### Excluir

Remove o insumo selecionado da composição.

### Configurações Avançadas

Permite configurar informações adicionais do insumo, como:

- Código da Variação
- Nome da Variação
- Nome do Item

Utilize esta opção quando houver necessidade de controlar variações específicas dentro da composição do produto.

## Dúvidas Frequentes

### Como criar uma ficha técnica?

Clique em **+ Novo**, selecione o produto final, informe os insumos necessários e clique em **Salvar**.

### Como adicionar um insumo?

Preencha os dados do item na seção de insumos e clique em **Add**.

### Como informar perdas de produção?

Utilize o campo **(% Perda)** para registrar o percentual esperado de quebra ou desperdício.

### Como atualizar os custos da ficha técnica?

Acesse **Menu > Recalcular Custo** e confirme a operação.

### Como exportar as fichas técnicas?

Acesse **Menu > Exportar** para gerar uma planilha Excel.

### Como remover todos os insumos da ficha?

Utilize **Menu > Excluir Todos os Insumos**.
