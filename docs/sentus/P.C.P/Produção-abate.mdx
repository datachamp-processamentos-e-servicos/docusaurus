# Produção Abate

## Visão Geral

A tela de Produção e Desmontagem é utilizada para controlar processos produtivos e operações de desmontagem dentro da empresa.

Por meio desta funcionalidade, é possível registrar produções, controlar o consumo de insumos, acompanhar recursos utilizados, realizar desmontagens e consultar informações relacionadas aos processos executados.

---

## Observações Importantes

- Todos os campos marcados com (*) são obrigatórios.
- Revise os dados informados antes de salvar.
- Verifique o depósito de consumo antes de concluir o processo.
- Utilize as observações para registrar informações importantes.
- Mantenha a situação dos registros atualizada para facilitar o acompanhamento das operações.
- Os recursos e insumos utilizados podem impactar diretamente os controles de estoque e produção.

---

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para criar um novo registro de Produção ou Desmontagem.

Preencha os campos obrigatórios e clique em **Salvar** para concluir o cadastro.

### Editar

Permite alterar informações do registro já cadastrado.

Caso exista campo de CPF/CNPJ no cadastro, não altere essa informação após a criação do registro, pois o sistema poderá considerar a alteração como um novo cadastro.

### Informações Cadastrais

#### Início

Informe a data de início da produção ou desmontagem.

#### Término

Informe a data de término do processo.

#### Participante

Selecione o participante relacionado à produção ou desmontagem.

#### Depósito Consumo

Informe o depósito de onde serão consumidos os itens utilizados no processo.

#### Observação

Utilize este campo para registrar informações complementares ou observações relevantes.

## Menu

### Visão Geral

Exibe todas as informações relacionadas ao registro selecionado.

### Faturar

Permite realizar o faturamento da produção ou desmontagem selecionada.

### Exportador

Gera um arquivo PDF contendo as informações do registro.

Ao exportar é possível definir:

#### Opções

Define o formato da exportação.

#### Produções

Permite exportar:

- Registros listados
- Registros selecionados

### Exportar

Utilize esta opção para gerar arquivos PDF contendo as informações exibidas na tela.

## Pesquisa Avançada

Utilize a Pesquisa Avançada para localizar registros específicos.

### Filtros Disponíveis

- Tipo
- Participante
- Observação
- Situação

### Filtro por Data

Permite consultar registros através de:

- Data Início
- Data Fim 

## Mais Ações

### Editar

Permite alterar as informações do registro.

### Produções

Exibe as produções vinculadas ao registro selecionado.

### Referenciar

Permite visualizar os insumos e produtos relacionados à produção realizada.

### Situação

Permite alterar a situação do processo.

As situações normalmente utilizadas são:

- Em Aberto
- Finalizada
- Cancelada

### Referenciamento

Gera o referenciamento da produção selecionada.

### Recursos

Permite visualizar os recursos utilizados durante o processo.

São exibidas informações como:

- Recurso utilizado
- Quantidade
- Unidade

### Visão Geral

Apresenta um resumo completo da produção ou desmontagem, incluindo produtos, insumos e informações relacionadas ao processo.

## Dúvidas Frequentes

### Como criar uma produção ou desmontagem?

Clique em **+ Novo**, preencha os campos obrigatórios e clique em **Salvar**.

### Como finalizar uma produção?

Acesse **Mais Ações > Situação** e altere o status para finalizada.

### Como visualizar os insumos utilizados?

Utilize a opção **Referenciar**.

### Como consultar os recursos utilizados?

Acesse **Mais Ações > Recursos**.

### Como gerar um relatório?

Acesse **Menu > Exportador** e gere o PDF desejado.

### Como localizar registros por período?

Utilize a Pesquisa Avançada e informe a Data Início e Data Fim.

### Como consultar informações completas do processo?

Utilize a opção **Visão Geral**.
