## Cadastrar uma Produção

Acesse:

**P.C.P. → Produção Abate**

1. Clique em **+ Novo** para iniciar uma nova produção;
2. Informe a **Data de Início**;
3. Selecione o **Participante** responsável pela produção;
4. Escolha o **Depósito de Consumo**, que será utilizado para baixar os insumos consumidos;
5. Se desejar, registre uma **Observação** sobre a produção;
6. Após concluir o processo produtivo, informe a **Data de Término**;
7. Clique em **Salvar** para registrar a produção.

## Informações da Produção

Nesta tela são registradas as informações gerais de uma produção.

Os dados informados serão utilizados para identificar a produção, controlar seu período de execução e definir de qual depósito os materiais serão consumidos.

Entre as principais informações estão:

- Data de início da produção;
- Data de término;
- Participante responsável;
- Depósito de consumo;
- Observações da produção.

## Observações

O campo **Observação** pode ser utilizado para registrar informações importantes sobre o processo produtivo, como ocorrências, instruções internas ou qualquer detalhe relevante para futuras consultas.

---

**Finalizando**

Após revisar as informações preenchidas, clique em **Salvar** para concluir o cadastro da produção.
