# OcorrenciasⓍ

:::note[Antes de começar]

- O preenchimento dos campos com **(*)** é obrigatório.
- Ao posicionar o cursor do mouse sobre um campo, o sistema exibe uma breve explicação sobre o preenchimento.

Acesse:

**P.C.P → Auxiliares → Ocorrências**

1. Clique em **+ Novo** para cadastrar uma ocorrência;
2. Informe o código e o nome da ocorrência;
3. Selecione o tipo da ocorrência;
4. Caso necessário, vincule uma **Ocorrência Pai**;
5. Informe uma observação para descrever a finalidade da ocorrência;
6. Adicione tags, se utilizadas pela empresa;
7. Defina a situação do cadastro;
8. Clique em **Salvar** para concluir.

## Informações da Ocorrência

Nesta seção são cadastradas as informações que identifcam e classificam a ocorrência utiliada nos processos produtivos.

Entre as informações disponíveis estão:

- Código;
- Nome;
- Tipo;
- Ocorrência Pai;
- Tags;
- Observação;
- Situação.

O campo **Tipo** define a categoria da ocorrência conforme a necessidade do processo.

A opção **Ocorrência Pai** permite criar uma relação hierárquica entre ocorrências, facilitando sua organização.

O campo **Tags** pode ser utilizado para classificar e localizar ocorrências com maior facilidade.

A **Observação** pode ser utilizada para registrar informações complementares sobre a utilização da ocorrência.

O campo **Situação** permite manter a ocorrência disponível para uso ou inativá-la quando não for mais necessária.

---

**Finalizando**

Após revisar todas as informações, clique em **Salvar** para concluir o cadastro da ocorrência.
