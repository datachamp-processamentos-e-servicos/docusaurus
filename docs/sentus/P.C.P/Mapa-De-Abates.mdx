## Cadastrar Novo Mapa de Abate

Acesse:

**P.C.P. → Mapa de Abates**

1. Clique em **+ Novo** para criar um novo mapa de abate;
2. Informe a data de emissão;
3. Selecione o **Tipo** de abate;
4. Preencha as informações de quantidade e peso conforme os dados da produção;
5. Caso necessário, registre observações;
6. Clique em **Salvar** para concluir o cadastro.

## Informações do Mapa de Abate

Nesta seção são registrados os dados utilizados para acompanhar o rendimento do processo de abate.

Entre as informações disponíveis estão:

- Tipo de abate;
- Situação;
- Aves abatidas;
- Aves condenadas;
- Aves mortas;
- Quantidade prevista;
- Peso previsto;
- Rendimento;
- Aproveitamento;
- Observações.

As quantidades e os pesos informados são utilizados para calcular automaticamente os indicadores de rendimento e aproveitamento do processo.

Os campos **Aves Condenadas** e **Aves Mortas** permitem registrar perdas ocorridas durante o abate, contribuindo para o acompanhamento dos índices produtivos.

O campo **Previsto** pode ser utilizado para comparar a quantidade planejada com o resultado efetivamente obtido.

A área de **Observação** pode ser utilizada para registrar informações importantes sobre o lote abatido, ocorrências ou qualquer detalhe relevante para consulta futura.

O campo **Situação** permite selecionar o status: Previsto, Em Aberto, Em Produção, Encerrado, Cancelado.

---

**Finalizando**

Após revisar todas as informações, clique em **Salvar** para concluir o cadastro do mapa de abate.
