# Gerar carta de correção de NFe
- Acesse Comercial → NFE;
- Localizar Nota Fiscal, para isso use a pesquisa rápida ou clique no ícone da lupa localizado no canto superior direito para realizar a busca;
- Em seguida, na linha correspondente à nota fiscal desejada, clique no botão Ações (+), depois clique em eventos e selecione a opção **Carta de Correção**;
- No campo Correção, informe as alterações que precisam ser realizadas e, após preencher as informações, clique em **Enviar**.

:::note[Observações Importantes]

- Apenas a última carta de correção é válida, ou seja, se for gerada uma segunda carta de correção a primeira deixa de valer;
-	A descrição da correção deve ser clara e objetiva;
-	O campo de correção exige no mínimo 15 caracteres;
- A Carta de Correção não pode ser utilizada para alterar informações que impactem:
  -	Valores fiscais;
  -	Base de cálculo dos impostos;
  -	Quantidade ou valor dos produtos;
  -	Dados do remetente ou destinatário que impliquem mudança de identificação;
  -	Data de emissão da nota fiscal.
