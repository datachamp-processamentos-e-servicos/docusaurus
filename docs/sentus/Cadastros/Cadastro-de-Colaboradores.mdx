# Colaboradores Ⓧ

# Cadastro de Colaboradores

:::note[Antes de começar]

- Todos os campos marcados com **(*)** são obrigatórios.
- Verifique se os dados do colaborador estão corretos antes de salvar o cadastro.

:::

## Como cadastrar

Acesse:

**Cadastros → Colaboradores**

1. Clique em **+ Novo** para iniciar um novo cadastro.
2. Preencha as informações solicitadas.
3. Complete os campos obrigatórios.
4. Informe os dados complementares do colaborador.
5. Clique em **Salvar** para concluir.

---

## Informações cadastrais

Nesta seção são informados os principais dados do colaborador, incluindo informações de identificação, dados fiscais e classificação do cadastro.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

---

## Endereço

Nesta seção deve ser informado o endereço principal do colaborador.

Caso necessário, utilize **Demais Endereços** para cadastrar outros endereços.

---

## Dados do colaborador

Nesta seção são definidas as informações específicas do colaborador, como supervisor e situação do cadastro.

Preencha apenas as informações necessárias para o cadastro.

---

## Contatos

Caso necessário, cadastre um ou mais contatos vinculados ao colaborador.

Preencha os campos desejados e clique em **Confirmar** para adicionar o contato.

---

## Finalizando

Após revisar as informações, clique em **Salvar** para concluir o cadastro.
