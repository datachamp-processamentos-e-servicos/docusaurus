## Cliente MEI

Clientes enquadrados como **MEI** devem possuir a configuração fiscal correta.

- Informe a **Inscrição Estadual**, quando existente.
- Defina corretamente o **Indicador de Inscrição Estadual**.
- Confira o **Regime Tributário** antes de salvar.

> Para mais detalhes, consulte o artigo **Como cadastrar um cliente MEI**.

## Contatos

Na aba **Contatos** é possível cadastrar diversas pessoas vinculadas ao cliente.

Cada contato pode possuir:

- Nome;
- Setor;
- Telefone;
- E-mail;
- Tipo de documento recebido;
- Tipo de anexo (PDF, XML, PDF + XML).

> Consulte o artigo **Como incluir contatos nos clientes**.

## Regras Comerciais

As regras comerciais permitem personalizar o comportamento das vendas para um cliente.

É possível configurar:

- Representante;
- Tabela de preços;
- Descontos;
- Comissão;
- Condições comerciais.

> Consulte o artigo **Parametrizar regras comerciais**.
