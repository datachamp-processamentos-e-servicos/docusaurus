# Clientes

:::note[Antes de começar]

- Todos os campos marcados com **(*)** são obrigatórios.
- Verifique se o CNPJ/CPF e os demais dados cadastrais estão corretos antes de salvar o cadastro.

:::

## Como cadastrar

Acesse:

**Cadastros → Clientes**

1. Clique em **+ Novo** para iniciar um novo cadastro.
2. Preencha as informações solicitadas.
3. Complete os campos obrigatórios.
4. Adicione os endereços e demais informações necessárias.
5. Clique em **Salvar** para concluir.

---

## Informações cadastrais

Nesta seção são informados os principais dados do cliente, como informações fiscais, identificação e classificação do cadastro.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

---

## Endereço

Nesta seção deve ser informado o endereço principal do cliente.

Caso necessário, utilize **Demais Endereços** para cadastrar outros endereços.

---

## Configurações do cliente

Nesta seção são definidas as configurações utilizadas nas operações realizadas com o cliente, como situação do cadastro, forma de pagamento e demais parâmetros relacionados.

Preencha apenas as informações necessárias para o cadastro.

---

## Dados do representante

Informe as informações específicas do representante, como supervisor, situação e demais configurações disponíveis.

Preencha os campos necessários, como:

- Supervisor;
- IRRF;
- Lote/Carga.

### Situação

Defina a situação do cadastro:

- *Ativo*: o representante poderá ser utilizado normalmente.
- *Bloqueado*: impede novas operações utilizando o representante.
- *Inativo*: mantém o cadastro apenas para consulta e histórico.

---

## Contatos

Caso necessário, cadastre um ou mais contatos vinculados ao cadastro.

Preencha os campos desejados e clique em **Confirmar** para adicionar o contato.

---

## Finalizando

Após revisar as informações, clique em **Salvar** para concluir o cadastro.
