# Clientes Ⓧ

:::note[Antes de começar]

- O cadastro de clientes é utilizado para registrar todas as informações necessárias para processos comerciais, fiscais e financeiros.
- Mantenha os dados sempre atualizados para garantir o correto funcionamento das vendas e emissões fiscais.

:::

## Como acessar

Acesse:

**Cadastros → Clientes**

1. Clique em **Novo** para cadastrar um cliente ou abra um registro existente.
2. Preencha as informações necessárias.
3. Clique em **Salvar** para concluir o cadastro.

---

## Informações Cadastrais

Nesta seção são registrados os dados principais do cliente.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

Entre as informações disponíveis estão:

- CNPJ/CPF;
- Nome e Nome Fantasia;
- Inscrição Estadual;
- Regime Tributário;
- Telefone e E-mail;
- Grupo e Subgrupo;
- Dados financeiros;
- Observações.

---

## Endereço

Informe o endereço principal do cliente, incluindo:

- Logradouro;
- Número;
- Bairro;
- CEP;
- Município;
- Complemento (quando necessário).

Caso o cliente possua outros endereços, utilize a aba **Demais Endereços** para cadastrá-los.

---

## Configurações do Cliente

Nesta seção podem ser definidas informações utilizadas nas operações comerciais.

Entre elas estão:

- Situação do cliente;
- Portador;
- Forma de pagamento;
- Operação tributária;
- Lote/Carga.

---

## Lista de Contatos

Cadastre os contatos vinculados ao cliente para facilitar a comunicação.

Entre as informações disponíveis estão:

- Nome;
- Telefone;
- E-mail;
- Setor;
- Aplicação;
- Anexos.

Após preencher os dados do contato, clique em **Confirmar** para adicioná-lo à lista.

---

## Finalizando

Após revisar todas as informações, clique em **Salvar** para concluir o cadastro do cliente.
