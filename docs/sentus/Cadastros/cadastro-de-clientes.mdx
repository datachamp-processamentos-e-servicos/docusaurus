## Cadastrar Novo Cliente

Acesse:

**Cadastros → Clientes**

1. Clique em **+ Novo** para cadastrar um cliente;
2. Selecionar o tipo de pessoa;
3. Informar o documento ou clicar no botão **Pesquisar Participante Existente** e pesquisar pelo nome, documento ou código de uma pessoa que já esteja cadastrada com outro papel no sistema, por exemplo, como fornecedor, como representante, como transportador ou como colaborador;
4. Clique em **Avançar**.

## Informações Cadastrais

Nesta seção são registrados os dados principais do cliente.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

- Campo Código é um campo de sua escolha, ele não pode ser repetido em dois clientes diferentes e pode ser preenchido com letras e números;
- Indicador de IE define o perfil de contribuinte do cliente, basicamente se o cliente possui Inscrição Estadual, selecionar "contribuinte" e caso o cliente não possua Inscrição estadual, selecionar "não contribuinte"; 
- Regime Tributário deve ser obrigatoriamente informado quando o cliente for MEI, caso contrário a Inscrição estadual de MEI não será aceita;
- O campo de **E-mail** não é usado para envio automático de documentos, para isso é necessário cadastrar contatos;
- Município dever ser selecionado da lista de pesquisa, não somente digitado.
- É possível cadastrar outros endereços para o mesmo cliente, clicar em "demais endereços"

CLIQUE EM AVANÇAR

## Configurações do Cliente

Nesta seção podem ser definidas informações utilizadas nas operações comerciais.
A seleção de informações nesses campos ajuda a definir os padrões de faturamento para esse cliente, mas podem ser editadas na emissão de um documento.

- Portador;
- Forma de pagamento;
- Operação tributária;
- Lote/Carga.

O campo situação permite bloquear ou inativar um cliente.

## Lista de Contatos

Cadastre um ou mais contatos vinculados ao cliente para facilitar a comunicação.
A principal função desses contatos é o envio automático de documentos por E-mail.

- Nome;
- Telefone (opcional);
- Setor (opcional);
- E-mail;
- Aplicação (define quais documentos emitidos, serão enviados para esse contato);
- Anexos (define quais documentos serão anexados no email para esse contato).

Após preencher os dados do contato, clique em **Confirmar** para adicioná-lo à lista de contatos.
Reforçando: Podem ser cadastrados vários contatos para cada cliente.

## Endereço

Informe o endereço principal do cliente, incluindo:

- Logradouro;
- Número;
- Bairro;
- CEP;
- Município;
- Complemento (quando necessário).

Caso o cliente possua outros endereços, utilize a aba **Demais Endereços** para cadastrá-los.
