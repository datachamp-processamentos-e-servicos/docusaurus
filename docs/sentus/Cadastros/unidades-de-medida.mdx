## Cadastrar Nova Unidade

Acesse:

**Cadastros → Itens → Unidades de Medida**

1. Clique em **+ Novo** para cadastrar uma unidade ou selecione uma unidade existente na lista para editá-la.
2. Preencha as informações necessárias;
3. Clique em **Salvar** para concluir o cadastro.

## Informações da Unidade

Nesta seção são definidas as informações utilizadas para identificar e configurar a unidade de medida dos itens cadastrados no sistema.

Conforme a necessidade da empresa, podem ser informadas informações como:

- Nome;
- Abreviatura;
- Tipo;
- Unidade P/FCI;
- Situação;
- Quantidade de Decimais;
- Largura;
- Altura;
- Profundidade;
- Cubagem;
- Peso;
- Observação.

O campo **Situação** permite selecionar o status : Ativo ou inativo.


Essas informações auxiliam na padronização das unidades de medida e podem ser utilizadas em processos comerciais, fiscais e logísticos.

---

**Finalizando**

Após revisar as informações, clique em **Salvar** para concluir o cadastro da unidade.
