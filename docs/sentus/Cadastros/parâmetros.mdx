## Configurar Regras e Parâmetros

Acesse:

**Cadastros → Demais Cadastros → Regras e Parâmetros**

1. Selecione o parâmetro desejado na lista.
2. Informe ou altere o valor conforme a necessidade.
3. Clique em **Salvar** para aplicar as alterações.

## Parâmetros

Nesta seção são exibidos os parâmetros disponíveis para configuração das funcionalidades do sistema.

Cada parâmetro possui um tipo, um valor configurável e uma ação correspondente.

Caso necessário, utilize a opção **Sincronização**, disponível no menu **Opções**, para atualizar as informações relacionadas aos parâmetros.

As alterações realizadas nessa tela podem impactar o funcionamento de determinadas rotinas do sistema.

---

**Finalizando**

Após revisar as informações, clique em **Salvar** para concluir as alterações dos parâmetros.
