# Representantes

:::note[Antes de começar]

- Todos os campos marcados com **(*)** são obrigatórios.
- O representante deve possuir um **CPF**, **CNPJ** ou **Documento de Estrangeiro** válido.
- Caso o representante já esteja cadastrado no sistema, utilize **Pesquisar Participante Existente** para aproveitar as informações já cadastradas.

:::

## Como cadastrar um representante

Acesse:

**Cadastros → Representantes**

1. Clique em **+ Novo**.
2. Selecione o **Tipo de Pessoa**:
   - Física;
   - Jurídica;
   - Estrangeiro.
3. Informe o **CPF**, **CNPJ** ou **Documento de Estrangeiro**.
4. Clique em **Avançar**.

---

## Informações cadastrais

Nesta etapa, preencha os dados básicos do representante.

Além dos campos de identificação e contato, configure as seguintes informações:

### Indicador de Inscrição Estadual

Define a situação fiscal do representante.

As opções disponíveis são:

- Contribuinte;
- Contribuinte Isento;
- Não Contribuinte;
- Contribuinte – Compra para Consumo Próprio;
- Não Contribuinte – Tratado como Contribuinte (MEI - Revenda);
- Contribuinte Isento – Tratado como Contribuinte (MEI - Revenda).

### Regime Tributário

Selecione o regime tributário correspondente ao representante.

Opções disponíveis:

- Simples Nacional;
- Simples Nacional – Excesso de Sublimite da Receita Bruta;
- MEI.

### Grupo

Selecione o grupo ao qual o representante pertence.

Por padrão, utiliza-se o grupo **Diversos**.

### Pessoa

Informe o tipo de pessoa do representante:

- Física;
- Jurídica;
- Estrangeiro.

---

## Finalizando o cadastro

Após clicar em **Avançar**, serão exibidas as informações específicas do representante.

Preencha os campos necessários, como:

- Supervisor;
- IRRF;
- Lote/Carga.

### Situação

Defina a situação do cadastro:

- **Ativo**: o representante poderá ser utilizado normalmente.
- **Bloqueado**: impede novas operações utilizando o representante.
- **Inativo**: mantém o cadastro apenas para consulta e histórico.

Após conferir as informações, clique em **Salvar** para concluir o cadastro.
