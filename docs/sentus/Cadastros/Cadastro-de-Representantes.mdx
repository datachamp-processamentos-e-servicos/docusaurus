# Representantes Ⓧ

:::note[Antes de começar]

- Todos os campos marcados com **(*)** são obrigatórios.
- O representante deve possuir um **CPF** ou **CNPJ** válido.
- Utilize **Pesquisar Participante Existente** para reaproveitar o cadastro.

:::

## Como cadastrar um representante

Acesse:

**Cadastros → Representantes**

1. Clique em **+ Novo**.
2. Informe o **CPF** ou **CNPJ** do representante.
3. Clique em **Avançar**.
4. Preencha os campos obrigatórios.
5. Clique em **Salvar** para concluir o cadastro.

---

## Informações cadastrais

Nesta seção, preencha as informações cadastrais do representante.

Configure os dados fiscais conforme a necessidade e informe os campos obrigatórios antes de prosseguir.

---

## Endereço

Preencha o endereço principal do representante.

Caso necessário, utilize a aba **Demais Endereços** para cadastrar endereços adicionais.

---

## Dados do representante

Informe as informações específicas do representante, como supervisor, situação e demais configurações disponíveis.

Preencha os campos necessários, como:

- Supervisor;
- IRRF;
- Lote/Carga.

### Situação

Defina a situação do cadastro:

- *Ativo*: o representante poderá ser utilizado normalmente.
- *Bloqueado*: impede novas operações utilizando o representante.
- *Inativo*: mantém o cadastro apenas para consulta e histórico.


---

## Lista de Contatos

Cadastre os contatos vinculados ao cliente para facilitar a comunicação.

Entre as informações disponíveis estão:

- Nome;
- Telefone;
- E-mail;
- Setor;
- Aplicação;
- Anexos.

Após preencher os dados do contato, clique em **Confirmar** para adicioná-lo à lista.

---

Após conferir as informações, clique em *Salvar* para concluir o cadastro.
