Cadastro de Representantes

## Visão Geral

A tela de Cadastro de Representantes é utilizada para cadastrar, consultar e gerenciar os representantes comerciais da empresa. 
Nela são armazenadas informações cadastrais, fiscais e comerciais que serão utilizadas nos processos de vendas, comissões e relacionamento comercial com os clientes.

---

## Principais Funcionalidades

### Novo Cadastro

Ao clicar em **+ Novo**, é possível cadastrar um novo representante informando os dados cadastrais obrigatórios.

Campos principais:

- CPF ou CNPJ;
- Nome ou Razão Social;
- Nome Fantasia;
- Inscrição Estadual;
- Regime Tributário;
- Telefone;
- E-mail;
- Data de Fundação ou Nascimento.

### Informações Cadastrais

O cadastro permite registrar:

- Código interno do representante;
- Indicador de Inscrição Estadual;
- Inscrição Estadual;
- Regime Tributário;
- Nome;
- Nome Fantasia;
- E-mail;
- Telefone;
- Fundação ou Nascimento;
- Inscrição Municipal.

### Consulta de Cadastro

Através da opção **Consultar Cadastro**, é possível obter informações complementares do CNPJ informado, auxiliando no preenchimento correto dos dados cadastrais.

---

## Atualização de Comissão

Permite atualizar a comissão dos representantes de forma centralizada.

É possível definir:

- Representante;
- Data Inicial;
- Data Final;
- Tipo de Movimentação;
- Atualização das comissões.

Essa funcionalidade é utilizada para revisar percentuais de comissão aplicados às vendas.

---

## Mensagem para Vendas

Permite enviar comunicados ou orientações para um ou mais representantes.

Informações disponíveis:

- Seleção do representante;
- Campo para digitação da mensagem.

Também permite:

- Exportar representantes para Excel;
- Exportar parâmetros (somente administradores do ERP).

---

## Pesquisa Avançada

A Pesquisa Avançada permite localizar representantes através de diversos filtros.

Filtros disponíveis:

- Código;
- CPF/CNPJ;
- Tags;
- Inscrição Estadual;
- Nome;
- Nome Fantasia;
- Supervisor;
- Município;
- Observações;
- Situação.

Também é possível ordenar os resultados por:

- Nome;
- Código;
- Grupo.

---

## Mais Ações

A opção **Mais Ações** reúne funcionalidades complementares para gerenciamento dos representantes.

Principais recursos disponíveis:

- Editar cadastro;
- Alterar situação;
- Configurar informações comerciais;
- Abrir conversa;
- Regras e parâmetros (somente administradores).

### Situação do Representante

Permite controlar o status do cadastro.

Situações disponíveis:

- Ativo;
- Inativo;
- Bloqueado.

### Informações Comerciais

Permite vincular informações comerciais ao representante.

É possível configurar:

- Tabelas de preço;
- Limites de desconto;
- Percentuais de comissão.

### Tabela de Preços

Define as condições comerciais que poderão ser utilizadas pelo representante.

Configurações disponíveis:

- Representante;
- Percentual máximo de desconto;
- Percentual de comissão.

### Abrir Conversa

Disponibiliza acesso rápido aos meios de contato do representante.

Recursos:

- Ligação telefônica;
- WhatsApp.

---

## Observações Importantes

- Todos os campos marcados com (*) são obrigatórios.
- Caso o representante possua Inscrição Estadual ativa, selecionar **Contribuinte** e informar a IE.
- Caso não possua IE, selecionar **Não Contribuinte**.
- Caso seja isento, selecionar **Isento**.
- O CEP deve ser informado apenas com números.
- Caso o endereço não possua número, informar **SN**.
- Alterações de CPF ou CNPJ caracterizam um novo cadastro.
