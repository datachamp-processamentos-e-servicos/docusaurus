# Representantes

:::note[Antes de começar]

- Todos os campos marcados com **(*)** são obrigatórios.
- O representante deve possuir um CPF, CNPJ ou documento de estrangeiro válido.
- Caso o representante já esteja cadastrado como participante, utilize a opção **Pesquisar Participante Existente** para aproveitar as informações já registradas.

:::

## Como cadastrar um representante

Acesse:

**Cadastros → Representantes**

1. Clique em **+ Novo**.
2. Selecione o **Tipo de Pessoa**:
   - Física;
   - Jurídica;
   - Estrangeiro.
3. Informe o **CPF**, **CNPJ** ou **Documento de Estrangeiro**.
4. Caso o participante já exista no sistema, utilize **Pesquisar Participante Existente**.
5. Clique em **Avançar**.

---

## Informações cadastrais

Preencha os dados do representante, como:

- Nome;
- Telefone;
- E-mail;
- Endereço;
- Grupo;
- Demais informações cadastrais.

### Indicador de IE

Escolha a opção correspondente à situação fiscal do representante:

- Contribuinte;
- Contribuinte Isento;
- Não Contribuinte;
- Contribuinte - Compra para Consumo Próprio;
- Não Contribuinte - Tratado como Contribuinte (MEI - Revenda);
- Contribuinte Isento - Tratado como Contribuinte (MEI - Revenda).

### Regime Tributário

Selecione o regime tributário do representante:

- Simples Nacional;
- Simples Nacional - Excesso de Sublimite da Receita Bruta;
- MEI.

### Grupo

Selecione o grupo desejado.

Por padrão, o cadastro utiliza o grupo **Diversos**.

### Pessoa

Defina o tipo de pessoa conforme o cadastro:

- Física;
- Jurídica;
- Estrangeiro.

---

## Finalizar o cadastro

Após clicar em **Avançar**, preencha as informações específicas do representante.

Os principais campos são:

- Supervisor;
- IRRF;
- Lote/Carga;
- Situação.

### Situação

Defina a situação do representante:

- Ativo;
- Bloqueado;
- Inativo.

Após preencher as informações, clique em **Salvar** para concluir o cadastro.

:::tip[Dica]

Caso o representante já exista como participante no sistema, utilize a pesquisa de participantes para evitar cadastros duplicados e reaproveitar as informações já cadastradas.

:::
