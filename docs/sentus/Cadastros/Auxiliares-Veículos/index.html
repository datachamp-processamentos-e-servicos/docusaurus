Veículos

Visão Geral​

A tela de Veículos é utilizada para realizar o cadastro e gerenciamento dos veículos utilizados pela empresa em suas operações diárias.

Por meio desta funcionalidade, é possível registrar informações de identificação, características físicas, capacidade de carga, proprietário e demais dados necessários para o controle da frota.

Observações Importantes​

  • Todos os campos marcados com (*) são obrigatórios.
  • O RENAVAM deve ser informado corretamente para identificação do veículo.
  • A situação do veículo influencia sua disponibilidade operacional.
  • Os campos de capacidade e tara auxiliam nos controles logísticos.
  • Tags podem ser utilizadas para facilitar pesquisas e identificações internas.
  • Mantenha as informações do proprietário sempre atualizadas.
  • O cadastro de novos veículos deve ser solicitado ao suporte quando aplicável.

Principais Funcionalidades​

Novo Cadastro​

Clique em + Novo para iniciar o cadastro de um novo veículo.

Preencha todos os campos obrigatórios identificados com (*).

Informe os dados do veículo conforme as informações do documento e do registro do veículo.

Após preencher todas as informações necessárias, clique em Salvar para concluir o cadastro.

Informações Cadastrais​

Nome​

Informe o nome utilizado para identificar o veículo no sistema.

Este campo facilita a localização e identificação do veículo durante as operações.

RENAVAM​

Informe o número do RENAVAM do veículo.

Este campo é utilizado para identificação oficial do veículo.

Situação​

Selecione a situação atual do veículo.

As opções podem incluir:

  • Ativo
  • Restrição
  • Bloqueado
  • Removido
  • Inativo

A situação determina se o veículo poderá ser utilizado nas operações da empresa.

Proprietário​

Selecione a empresa proprietária do veículo.

Este campo identifica a qual empresa o veículo está vinculado.

Placa​

Informe a placa do veículo.

Esta informação é utilizada para identificação operacional e documental.

UF​

Selecione o estado onde o veículo está registrado.

Tipo Veículo​

Defina o tipo do veículo.

As opções disponíveis são:

  • Tração
  • Reboque

Esta configuração identifica a função do veículo dentro da operação.

Tipo Rodado​

Selecione o tipo de rodado do veículo.

As opções podem incluir:

  • Truck
  • Toco
  • Cavalo Mecânico
  • Van
  • Utilitário
  • Outros

Esta informação auxilia na classificação da frota.

Carroceria​

Selecione o tipo de carroceria do veículo.

As opções podem incluir:

  • Não Aplicável
  • Aberta
  • Fechada/Baú
  • Graneleira
  • Porta Container
  • Sider

Esta configuração identifica as características estruturais do veículo.

Tara​

Informe o peso do veículo descarregado.

Este campo é utilizado para cálculos logísticos e controle de carga.

Cap. KG​

Informe a capacidade máxima de carga do veículo em quilogramas.

Cap. M3​

Informe a capacidade volumétrica do veículo em metros cúbicos.

TAGs / Veículos Relacionados​

Informe códigos, identificadores ou referências utilizadas para localizar o veículo.

Este campo pode ser utilizado para facilitar pesquisas e integrações operacionais.

Observação​

Informe observações relevantes sobre o veículo.

Utilize este campo para registrar informações adicionais que possam auxiliar no gerenciamento da frota.

Demais Funcionalidades​

Controle de Frota​

O cadastro de veículos permite manter informações atualizadas da frota utilizada pela empresa.

Os dados cadastrados podem ser utilizados em processos operacionais, logísticos e administrativos.

Pesquisa Avançada​

Clique no ícone de Pesquisa Avançada para localizar veículos específicos cadastrados no sistema.

Preencha um ou mais filtros e clique em Pesquisar para exibir os resultados correspondentes.

Filtros Disponíveis​

Nome​

Permite localizar veículos através do nome cadastrado.

Placa​

Permite localizar veículos através da placa.

Proprietário​

Permite localizar veículos através da empresa proprietária.

Tags​

Permite localizar veículos através de tags ou identificadores cadastrados.

Observação​

Permite localizar veículos através das informações registradas no campo de observação.

Tipo Veículo​

Permite localizar veículos conforme o tipo selecionado.

As opções disponíveis incluem:

  • Tração
  • Reboque

Situação​

Permite localizar veículos conforme sua situação cadastral.

As opções podem incluir:

  • Ativo
  • Restrição
  • Bloqueado
  • Removido
  • Inativo

Editar​

Localize o veículo desejado na listagem.

Clique na opção de edição disponível.

Realize as alterações necessárias.

Clique em Salvar para confirmar as modificações.

Dúvidas Frequentes​

Não consigo salvar o cadastro​

Verifique se todos os campos obrigatórios identificados com (*) foram preenchidos corretamente.

Confirme também se os dados obrigatórios do veículo foram informados.

Como cadastrar um novo veículo?​

Clique em + Novo.

Preencha as informações do veículo.

Clique em Salvar para concluir o cadastro.

Como pesquisar um veículo específico?​

Clique em Pesquisa Avançada.

Informe um ou mais filtros.

Clique em Pesquisar para visualizar os resultados.

Como alterar os dados de um veículo?​

Localize o veículo desejado.

Clique na opção de edição.

Realize as alterações necessárias.

Clique em Salvar para concluir.

Posso localizar um veículo pela placa?​

Sim.

Utilize o filtro Placa na Pesquisa Avançada para localizar rapidamente o veículo desejado.

Como cadastrar um novo veículo que não existe no sistema?​

Entre em contato com o suporte para solicitar a inclusão de novos veículos quando necessário.
