# Fornecedores

:::note[Antes de começar]

- Todos os campos marcados com **(*)** são obrigatórios.
- Verifique se os dados do fornecedor estão corretos antes de salvar o cadastro.

:::

## Como cadastrar

Acesse:

**Cadastros → Fornecedores**

1. Clique em **+ Novo** para iniciar um novo cadastro.
2. Preencha as informações solicitadas.
3. Complete os campos obrigatórios.
4. Informe os dados complementares do fornecedor.
5. Clique em **Salvar** para concluir.

---

## Informações cadastrais

Nesta seção são informados os principais dados do fornecedor, incluindo informações de identificação, dados fiscais e classificação do cadastro.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

---

## Endereço

Nesta seção deve ser informado o endereço principal do fornecedor.

Caso necessário, utilize **Demais Endereços** para cadastrar outros endereços.

---

## Dados do fornecedor

Nesta seção são definidas as informações utilizadas nas operações com o fornecedor, como situação do cadastro e forma de pagamento.

Preencha apenas as informações necessárias para o cadastro.

---

## Contatos

Caso necessário, cadastre um ou mais contatos vinculados ao fornecedor.

Preencha os campos desejados e clique em **Confirmar** para adicionar o contato.

---

## Finalizando

Após revisar as informações, clique em **Salvar** para concluir o cadastro.
