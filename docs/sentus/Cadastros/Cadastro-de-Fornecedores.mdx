# Fornecedores

## Visão Geral

A tela de Fornecedores é utilizada para cadastrar e gerenciar empresas ou pessoas que fornecem produtos e serviços para sua empresa.

Por meio desta funcionalidade, é possível manter informações cadastrais, endereços, contatos, dados fiscais e informações complementares necessárias para os processos de compras e relacionamento com fornecedores.

---

## Observações Importantes

- Todos os campos marcados com (*) são obrigatórios.
- O CPF/CNPJ não deve ser alterado em um cadastro existente.
- Caso o fornecedor não possua número no endereço, informe SN.
- O CEP deve ser preenchido apenas com números.
- Para endereços internacionais, o CEP pode ser preenchido com zeros.
- A Inscrição Estadual deve ser informada apenas com números.
- Algumas funcionalidades são restritas a usuários administradores.
- Cada contato deve possuir apenas um endereço de e-mail para envio de documentos.

---

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para cadastrar um novo fornecedor.

Preencha os campos obrigatórios marcados com (*) e informe os dados cadastrais necessários.

Após concluir o preenchimento, clique em **Salvar**.

### Informações Cadastrais

Nesta seção são registradas os principais dados do fornecedor.

#### Identificação

Informe os dados básicos do cadastro:

- CPF ou CNPJ
- Código Interno
- Nome ou Razão Social
- Nome Fantasia
- E-mail
- Telefone
- Data de Fundação ou Nascimento

#### Informações Fiscais

Configure os dados fiscais do fornecedor.

Os principais campos são:

- Indicador de Inscrição Estadual
- Inscrição Estadual
- Inscrição Municipal
- Regime Tributário
- Suframa

#### Classificação

Permite organizar os fornecedores através de:

- Grupo
- Subgrupo
- Tags

#### Observações

Utilize os campos de observações para registrar informações adicionais ou financeiras relacionadas ao fornecedor.

### Endereço Principal

Informe os dados do endereço principal do fornecedor.

Os principais campos são:

- Logradouro
- Número
- Complemento
- Bairro
- CEP
- Município
- Coordenadas (Latitude e Longitude)

### Demais Endereços

Permite cadastrar endereços adicionais para diferentes finalidades.

Os tipos disponíveis podem incluir:

- Endereço de Cobrança
- Endereço de Entrega
- Endereço de Coleta
- Outros

Também é possível informar observações e dados complementares relacionados ao endereço.

### Lista de Contratos

Permite cadastrar contatos específicos do fornecedor.

Para cada contato é possível informar:

- Nome
- Telefone
- Setor
- E-mail
- Aplicação
- Tipo de Anexo

Essas informações são utilizadas para o envio de documentos e notificações.

## Menu

### Exportar

Clique em **Menu > Exportar** para gerar uma planilha Excel contendo os fornecedores exibidos na tela.

O download será realizado automaticamente após a geração do arquivo.

## Pesquisa Avançada

Utilize a Pesquisa Avançada para localizar fornecedores específicos.

Informe um ou mais filtros e clique em **Pesquisar**.

### Filtros Disponíveis

- Código
- CPF-CNPJ
- Tags
- Inscrição Estadual
- Nome
- Nome Fantasia
- Município
- Observação
- Situação

### Ordenação

Os resultados podem ser organizados por:

- Nome
- Código
- Grupo

## Mais Ações

### Editar

Permite alterar informações previamente cadastradas do fornecedor.

### Item x Fornecedor(es)

Permite consultar quais produtos ou itens estão vinculados ao fornecedor selecionado.

### Contas

Disponíveis apenas para usuários com permissões administrativas.

### Distâncias

Permite visualizar informações de distância relacionadas ao fornecedor e aos locais de entrega.

### Regras e Parâmetros

Disponível apenas para usuários com permissões administrativas.

### Informações Adicionais

Permite registrar informações complementares relacionadas ao fornecedor.

### Abrir Conversa

Exibe os contatos disponíveis do fornedor e permite iniciar uma conversa diretamente pelo WhatsApp.

### Histórico

Permite consultar alterações realizadas no cadastro através do registro de alterações do sistema.

## Dúvidas Frequentes

### Não consigo salvar o cadastro

Verifique se todos os campos obrigatórios marcados com (*) foram preenchidos.

Confirme também se CPF/CNPJ, endereço e informações fiscais foram informados corretamente.

### Como consultar a Inscrição Estadual?

A consulta pode ser realizada através do portal do SINTEGRA.

Verifiqe se a inscrição está ativa antes de concluir o cadastro.

### O que informar no Indicador de IE?

- Contribuinte: quando o fornecedor possui Inscrição Estadual ativa.
- Não Contribuinte: quando não possui Inscrição Estadual.
- Isento: quando o fornecedor é isento de Inscrição Estadual.

### O fornecedor não possui número no endereço. O que informar?

Preencha o campo Número com **SN**.

### Como cadastrar mais de um e-mail para recebimento de documentos?

Cadastre contatos adicionais na seção **Lista de Contatos**.

Cada contato deve possuir apenas um endereço de e-mail.

### Como localizar um fornecedor já cadastrado?

Utilize a **Pesquisa Avançada** e informe filtros como CPF/CNPJ, nome, município ou situação.

### Como exportar a lista de fornecedores?

Acesse **Menu > Exportar** para gerar uma planilha Excel com os registros exibidos.
