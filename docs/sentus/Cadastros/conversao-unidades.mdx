## Cadastrar Nova Conversão de Unidade

Acesse:

**Cadastros → Itens → Conversão de Unidades**

1. Clique em **+ Novo** para cadastrar uma conversão de unidade ou selecione uma conversão existente na lista abaixo para editá-la;
2. Preencha as informações necessárias;
3. Clique em **Salvar** para concluir o cadastro.

## Informações da Conversão

Nesta seção são definidas as regras utilizadas para realizar a conversão entre diferentes unidades de medida.

Informe as configurações necessárias, como:

- Unidade de Origem;
- Unidade de Destino;
- Indicador do Fator;
- Fator de Conversão.

Essas informações permitem que o sistema realize automaticamente a conversão entre as unidades cadastradas durante as operações.

---

**Finalizando**

Após revisar as informações, clique em **Salvar** para concluir o cadastro da conversão de unidade.
