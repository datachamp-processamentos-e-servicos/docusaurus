- Alguns relatórios podem possuir filtros específicos para refinar os resultados apresentados.

## Visualizar um Relatório

Acesse:

**Relatórios**

1. Localize o relatório desejado na lista ou utilize o campo de pesquisa para encontrá-lo.
2. Clique no ícone de **Configuração** correspondente ao relatório que deseja visualizar.
3. Informe os filtros disponíveis, quando houver.
4. Clique em **Visualizar** para gerar o relatório.

## Filtros do Relatório

Alguns relatórios disponibilizam filtros para refinar as informações exibidas.

Os filtros variam conforme o relatório selecionado e podem incluir informações como:

- Situação;
- Grupo;
- Datas;
- Participantes;
- Entre outros.

O campo Situação permite selecionar um dos seguintes status: None (Nada/Nenhum), Ativo, Bloqueado ou Inativo.

Após configurar os filtros desejados, clique em **Visualizar** para gerar o relatório.

## Visualização e Exportação

Após a geração do relatório, as informações serão apresentadas na tela.

Também é possível:

- Exportar o relatório em **PDF**;
- Utilizar a opção **Outros Formatos**, quando disponível;
- Imprimir ou salvar o relatório conforme a necessidade.
