# Cadastro de Grupo de Itens

## Visão Geral

A tela de **Grupo de Itens** é utilizada para organizar os produtos cadastrados no sistema através de grupos e subgrupos. 
Essa estrutura facilita a organização do estoque, consultas, relatórios e processos comerciais, permitindo uma melhor categorização dos itens.

---

## Observações Importantes

- Todos os campos marcados com (*) são obrigatórios.
- Utilize nomes claros para facilitar a identificação dos grupos.
- Sempre vincule corretamente os subgrupos ao Grupo Pai correspondente.
- Mantenha a estrutura de grupos organizada para facilitar pesquisas e relatórios.
- Revise a categoria selecionada antes de salvar o cadastro.

---

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para cadastrar um novo grupo de itens.

Durante o cadastro, informe os dados necessários para identificar e organizar corretamente os itens vinculados ao grupo.

Campos disponíveis:
- Nome;
- Situação;
- Grupo Pai;
- Tipo;
- Categoria;
- Margem;
- Observação.

### Nome

Informe a nomenclatura que será utilizada para identificar o grupo de itens.

### Situação

Selecione a situação do grupo.

Opções comuns:

- Ativo;
- Inativo;
- Bloqueado.

### Grupo Pai

Caso o grupo seja um subgrupo, selecione o grupo principal ao qual ele pertence.

Essa funcionalidade permite criar uma estrutura hierárquica de grupos e subgrupos.

### Tipo

Defina o tipo do grupo.

Opções disponíveis:

- Diversos;
- Comercial;
- Subgrupo;
- Estatística.

### Categoria

Selecione a categoria correspondente ao grupo.

As categorias disponíveis podem variar conforme a configuração do sistema.

Exemplos:

- Diversos;
- Não entra em inventário.

### Margem

Informe a margem de lucro padrão que será utilizada como referência para os itens vinculados ao grupo.

### Observação

Utilize este campo para registrar informações complementares sobre o grupo de itens.

---

## Menu

### Exportar PDF

Clique em **Menu > Exportar PDF** para gerar um relatório em PDF contendo as informações exibidas na tela principal.

### Exportar XLS

Clique em **Menu > Exportar XLS** para exportar os dados para uma planilha Excel.

---

## Pesquisa Avançada

Clique na pesquisa avançada para localizar grupos específicos através de filtros.

Filtros disponíveis:

- ID;
- Nome;
- Tipo; 
- Categoria;
- Situação.

### Pesquisa por ID

Localiza o grupo através do código de itentificação interno.

### Pesquisa por nome

Permite localizar grupos através do nome cadastrado.

### Pesquisa por Tipo

Utilizada para localizar grupos conforme o tipo configurado.

### Pesquisa por Categoria

Permite filtrar os grupos pela categoria cadastrada.

### Pesquisa por Situação

Permite visualizar apenas grupos:

- Ativos;
- Inativos;
- Bloqueados.

---

## Dúvidas Frequentes

### Qual a diferença entre grupo e Subgrupo?

O grupo é a caregoria principal utilizada para organizar os itens.

o Subgrupo é uma subdivisão do grupo principal, permitindo uma organização mais detalhada dos produtos.

### Quando devo utilizar o Grupo Pai ?

Utilize o Grupo Pai quando estiver criando um subgrupo que pertença a outro grupo já cadastrado.

### A margem informada altera automaticamente os preços dos produtos?

A margem funciona como referência para os itens vinculados ao grupo, porém sua aplicação depende das configurações e processos utilizados pela empresa.

### Como exportar a lista de grupos?

1. Acesse o menu superior.
2. Clique em **Exportar PDF** ou **Exportar XLS**.
3. Aguarde a geração do arquivo.
4. O download será iniciado automaticamente.
