# Grupo de Itens Ⓧ

# Cadastro de Grupo de Itens

:::note[Antes de começar]

- Informe um nome que identifique o grupo de itens.
- Preencha as informações conforme a organização utilizada pela empresa.

:::

## Como cadastrar

Acesse:

**Cadastros → Itens → Grupo de Itens**

1. Clique em **+ Novo** para iniciar um novo cadastro.
2. Preencha as informações solicitadas.
3. Se necessário, vincule o grupo a um **Grupo Pai**.
4. Revise as informações informadas.
5. Clique em **Salvar** para concluir.

---

## Informações do grupo

Nesta seção são definidas as informações que identificam e classificam o grupo de itens, permitindo organizar os produtos de acordo com a estrutura utilizada pela empresa.

Preencha apenas as informações necessárias para o cadastro.

---

## Finalizando

Após revisar as informações, clique em **Salvar** para concluir o cadastro.
