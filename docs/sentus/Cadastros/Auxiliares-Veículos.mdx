# VeículosⓍ

:::note[Antes de começar]

- Verifique se o proprietário do veículo já está cadastrado no sistema.
- Preencha as informações do veículo conforme o documento de registro.

:::

## Como cadastrar

Acesse:

**Cadastros → Auxiliares → Veículos**

1. Clique em **+ Novo** para iniciar um novo cadastro.
2. Preencha as informações solicitadas.
3. Informe os dados do veículo e do proprietário.
4. Revise as informações cadastradas.
5. Clique em **Salvar** para concluir.

---

## Informações do veículo

Nesta seção são informados os dados de identificação e as características do veículo, como proprietário, placa, capacidade, tipo e demais informações utilizadas nas operações de transporte.

Preencha apenas as informações necessárias para o cadastro.

---

## Finalizando

Após revisar as informações, clique em **Salvar** para concluir o cadastro.
