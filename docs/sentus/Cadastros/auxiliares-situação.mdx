## Cadastrar Nova Situação

Acesse:

**Cadastros → Situações**

1. Clique em **+ Novo** para cadastrar uma situação ou selecione uma situação existente na lista abaixo para editá-la;
2. Preencha as informações necessárias;
3. Clique em **Salvar** para concluir o cadastro.

## Informações da Situação

Nesta seção são definidas as informações utilizadas para identificar e configurar uma situação no sistema.

Informe as informações necessárias, como:

- Nome;
- Aplicação;
- Situação;
- Observação.

O campo Situação permite selecionar o status: Ativo, restrição, bloqueado, removido e inativo.

Essas informações determinam onde a situação poderá ser utilizada e qual será seu comportamento dentro do sistema.

## Permissão - Grupos de Usuários

Nesta seção é possível definir permissões específicas para grupos de usuários.

Caso necessário, selecione o Nome desejado, defina o status correspondente (Liberado ou Bloqueado) e clique em **+ Add** para adicioná-lo à lista de permissões.

---

**Finalizando**

Após revisar as informações, clique em **Salvar** para concluir o cadastro da situação.
