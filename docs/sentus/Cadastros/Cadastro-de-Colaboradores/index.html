Colaboradores

Para um resumo em Slides Clique Aqui

Antes de começar
  • O preenchimento dos campos com (*) é obrigatório.
  • Ao posicionar o cursor do mouse sobre um campo, o sistema exibe uma breve explicação sobre o preenchimento.

Cadastrar Novo Colaborador​

Acesse:

Cadastros → Colaboradores

  1. Clique em + Novo para cadastrar um colaborador;
  2. Selecione o tipo de pessoa;
  3. Informe o documento ou clique em Pesquisar Participante Existente para localizar uma pessoa já cadastrada com outro papel no sistema, como cliente, fornecedor, representante ou transportadora;
  4. Clique em Avançar.

Informações Cadastrais​

Nesta seção são registrados os dados principais do colaborador.

Preencha os campos necessários, lembrando que os campos com (*) são obrigatórios.

  • O campo Código é de livre escolha, não pode ser repetido entre colaboradores e aceita letras e números;
  • O Indicador de IE define se o colaborador possui Inscrição Estadual. Se possuir, selecione Contribuinte; caso contrário, selecione Não Contribuinte;
  • O Regime Tributário deve ser informado obrigatoriamente quando o colaborador for MEI, caso contrário a Inscrição Estadual não será aceita;
  • O campo E-mail não é utilizado para o envio automático de documentos. Para isso, é necessário cadastrar contatos;
  • O Município deve ser selecionado na lista de pesquisa, e não apenas digitado;
  • Caso necessário, é possível cadastrar outros endereços na opção Demais Endereços.

Após preencher as informações, clique em Avançar.

Configurações do Colaborador​

Nesta seção podem ser definidas informações utilizadas nas operações comerciais.

As informações configuradas servirão como padrão para as operações do colaborador, mas poderão ser alteradas durante a emissão dos documentos.

Entre as configurações disponíveis estão:

  • Supervisor;
  • Situação.

O campo Situação permite bloquear ou inativar o colaborador.

Lista de Contatos​

Cadastre um ou mais contatos vinculados ao colaborador para facilitar a comunicação.

A principal função desses contatos é o envio automático de documentos por e-mail.

Informe os campos necessários, como:

  • Nome;
  • Telefone (opcional);
  • Setor (opcional);
  • E-mail;
  • Aplicação (define quais documentos serão enviados para esse contato);
  • Anexos (define quais documentos serão anexados ao e-mail).

Após preencher os dados do contato, clique em Confirmar para adicioná-lo à lista.

É possível cadastrar vários contatos para o mesmo colaborador.

Finalizando

Após revisar todas as informações, clique em Salvar para concluir o cadastro do colaborador.
