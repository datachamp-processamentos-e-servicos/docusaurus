## Inutilizar Numeração

Acesse:

**DFe(s) → Inut.Numeração**

1. Clique em **+ Novo** para iniciar uma inutilização;
2. Informe os dados da numeração que será inutilizada;
3. Preencha a justificativa da inutilização;
4. Clique em **Inutilizar** para enviar a solicitação.

## Informações da Inutilização

Nesta tela são informados os dados necessários para inutilizar uma faixa de numeração fiscal.

Preencha os principais campos, como:

- Emitente;
- Modelo do documento;
- Série;
- Data de Emissão;
- Número Inicial;
- Número Final;
- Justificativa;
- Status do DF-e.

A justificativa deve possuir, no mínimo, **15 caracteres**, conforme exigência da SEFAZ.

---

**Finalizando**

Após preencher as informações, clique em **Inutilizar** para enviar a solicitação de inutilização da faixa de numeração.
