## Consultar DFe(s)

Acesse:

**DFe(s) → Repositorório DFe(s)**

1. Acesse a lista de documentos fiscais eletrônicos;
2. Utilize a pesquisa para localizar um documento, se necessário;
3. Clique nas ações disponíveis para visualizar ou realizar operações sobre o DFe.

## Informações da Consulta

Nesta tela são exibidos os documentos fiscais eletrônicos processados pelo sistema.

As principais informações apresentadas são:

- Número / Modelo / Série;
- Data de Emissão;
- Nome do Emitente;
- Destinatário/Remetente;
- Chave de Acesso;
- NSU;
- Manifestação.

Na coluna **Ações** é possível:

- Visualizar as informações do documento;
- Imprimir o DANFE, quando disponível;
- Realizar o download do arquivo XML.

---

**Finalizando**

Utilize os filtros de pesquisa e as ações disponíveis para consultar e gerenciar os documentos fiscais eletrônicos.
