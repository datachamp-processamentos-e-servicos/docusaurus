## Consultar Lote DFe

Acesse:

**DFe(s) → Lote DFe(s)**

1. Selecione um lote da lista para visualizar suas informações, ou clique em **+ Novo** para criar um;
2. Consulte os dados de transmissão e processamento;
3. Clique em **Salvar** caso realize alguma alteração.

## Informações do Lote

Nesta seção são apresentados os dados do processamento do lote.

Os principais campos são:

- Data de Transmissão;
- Data de Processamento;
- Status;
- Recibo;
- Código da Mensagem;
- Motivo;
- Mensagem.

Essas informações permitem acompanhar o retorno e a situação do lote enviado.

---

**Finalizando**

Após consultar ou atualizar as informações, clique em **Salvar** para concluir.
