## Cadastrar uma Ordem de Serviço

Acesse:

**Serviços → Ordem de Serviço**

1. Clique em **+ Novo** para cadastrar uma nova ordem de serviço ou selecione uma existente para edição.
2. Preencha as informações principais da ordem.
3. Adicione os itens que serão executados.
4. Registre os eventos, quando necessário.
5. Clique em **Salvar**.

## Dados da Ordem de Serviço

Nesta seção são informadas as principais informações da ordem.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

Principais informações:

- Emissão.
- Responsável.
- Cliente.
- Localização.
- Prioridade.
- Situação.
- Tags.
- Contato.
- Observação.
- Descrição.

## Itens

Adicione os produtos ou serviços que fazem parte da ordem.

Informe:

- Referência.
- Item.
- Quantidade.
- Valor Unitário.

Após preencher as informações, clique em **+ Add** para incluir o item.

## Eventos

Nesta seção é possível registrar o andamento da ordem de serviço.

Os principais tipos de evento são:

- Novo Evento.
- Atendimento Presencial.
- Atividade.

Esses registros auxiliam no acompanhamento da execução da ordem.

**Finalizando**

Após conferir todas as informações, clique em **Salvar** para concluir o cadastro da Ordem de Serviço.
