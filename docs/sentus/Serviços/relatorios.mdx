## Emitir o Relatório

Acesse:

**Serviços → Relatórios**

1. Selecione o **Emitente**.
2. Informe o período em **Data Inicial** e **Data Final**.
3. Preencha os filtros desejados.
4. Clique em **Visualizar** para gerar o relatório.

## Filtros Disponíveis

Os principais filtros são:

- Emitente.
- Data Inicial.
- Data Final.
- Indicador de Emissão.
- Portador.
- Participante.
- Regime Tributário.
- Centro de Custos.

Utilize apenas os filtros necessários para facilitar a localização das informações.

## Visualização

Após informar os filtros desejados, clique em **Visualizar** para emitir o relatório das Notas de Serviço agrupadas por participante.

**Finalizando**

Caso necessário, altere os filtros e gere o relatório novamente para obter o resultado desejado.
