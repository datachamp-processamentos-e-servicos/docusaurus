## Emitir um RPS

Acesse:

**Serviços → Auxiliares → RPS**

1. Clique em **+ Novo** para iniciar um novo RPS.
2. Preencha as informações de cada etapa.
3. Clique em **Salvar** para gravar o documento.

## Dados do RPS

Nesta etapa são informadas as configurações gerais do recibo.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

**Prestador**

Os dados do prestador são exibidos automaticamente conforme a empresa selecionada.

**Documento**

Informe:

- Modelo.
- Série.
- Número.
- Data de Emissão.
- Status.

**Configurações do RPS**

Configure as informações fiscais do documento:

- Indicador ISSQN.
- Regime Tributário.
- Incentivador Fiscal.
- Simples Nacional.
- Tipo.
- Responsabilidade pela Retenção.
- Número do Processo (quando aplicável).

**Documento Substituído**

Quando o RPS substituir outro documento, informe:

- Série.
- Número.
- Tipo.

## Tomador do Serviço

Selecione o tomador através da pesquisa por:

- Nome.
- CNPJ/CPF.
- Código.

Também serão apresentados os municípios de:

- Prestação do serviço.
- Incidência.

## Serviço(s)

Cadastre os serviços que serão incluídos no RPS.

Informe:

- Serviço.
- Discriminação.
- Item da Lista de Serviço.
- Tipo de Serviço.
- CNAE.
- Quantidade.
- Valor Unitário.
- Valor dos Serviços.
- Descontos.
- Valor das Deduções.
- Base de Cálculo.
- Alíquota do ISS.
- Valor do ISS.
- ISS Retido.
- Situação Tributária.
- Valor Líquido.

Caso existam retenções, utilize a seção **Retenções** para adicioná-las.

## Dados da Obra

Quando o serviço estiver vinculado a uma obra, informe:

- Origem do endereço da obra.
- Logradouro.
- Número.
- Complemento.
- Bairro.
- CEP.
- Município.
- Número da ART.
- Número do CNO/CEI.
- Número da Matrícula.
- Número do Projeto.

## Cobrança

Configure as informações financeiras do documento.

Informe:

- Portador.
- Forma de Pagamento.
- Número da Fatura.
- Valor Original.
- Valor do Desconto.
- Valor Líquido.

As duplicatas geradas serão apresentadas na seção **Duplicata(s)**.

## Outras Informações

Caso necessário, informe:

- Dados do Intermediário.
- Informações adicionais de interesse do Fisco.
- Tags.

**Finalizando*8

Após revisar todas as informações, clique em **Salvar** para concluir o cadastro do RPS.
