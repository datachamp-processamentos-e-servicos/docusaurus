# Notas Terceiros Ⓧ

:::note[Antes de começar]

- O preenchimento dos campos com **(*)** é obrigatório.
- Ao posicionar o cursor do mouse sobre um campo, o sistema exibe uma breve explicação sobre o preenchimento.

<div style={{ textAlign: 'center' }}>
  <img
    src="/docusaurus/img/cadastros/servicos.png"
    alt="Cadastro de RPS"
    width="700"
  />
</div>

:::

Acesse:

**Serviços →Auxiliares → Notas Terceiros**

1. Clique em **+ Novo** para iniciar uma nova NFSe/DPS.
2. Preencha as informações de cada etapa.
3. Clique em **Salvar** para concluir a emissão.

## Dados Gerais

Nesta etapa são informadas as configurações iniciais da nota fiscal.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

**Documento**

Informe:

- Modelo.
- Série.
- Número.
- Complemento.
- Data de Emissão.
- Data/Hora de Entrada/Saída.
- Competência.
- Chave.
- Situação.
- Operação Tributária.
- CFOP.
- Operação Contábil.
- Centro de Custo.

**Tomador/Prestador**

Selecione o participante através da pesquisa por:

- Nome.
- CNPJ/CPF.
- Código.

Também serão apresentados os municípios de:

- Prestação do Serviço.
- Incidência.

**Detalhes**

Configure as informações complementares da nota:

- Representante.
- Finalidade.
- Indicador do Destinatário.
- Indicador ISSQN.

**Declaração Provisória de Serviços**

Caso a NFSe seja emitida com base em uma DPS, informe:

- Série.
- Número.
- Data de Emissão.

## Serviços

Cadastre os serviços que compõem a nota.

Informe:

- Serviço.
- Discriminação.
- Código da Operação.
- Tipo de Serviço.
- CNAE.
- Quantidade.
- Valor Unitário.
- Valor dos Serviços.
- Descontos.
- Valor das Deduções.
- Base de Cálculo do ISSQN.
- Alíquota do ISSQN.
- Valor do ISSQN.
- Percentual de Redução.
- Retenção.
- Valor Líquido.

Após preencher os valores, utilize **Calcular Tributação** quando necessário.

## Tributação

Nesta etapa são informadas as regras de tributação da Reforma Tributária.

Configure, quando aplicável:

- IBS/CBS.
- Base de Cálculo.
- CST.
- Classificação.
- Indicador de Doação.
- IBS para UF.
- IBS para Município.
- CBS.
- Redução.
- Devolução.
- Diferimento.
- Tributação Regular.
- Compra Governamental.
- Crédito Presumido da Operação.
- Crédito Presumido IBS.
- Crédito Presumido CBS.
- Estorno de Crédito.

## Cobrança

Informe os dados financeiros da nota.

Configure:

- Portador.
- Forma de Pagamento.
- Grupo.
- Número da Fatura.
- Valor Original.
- Valor do Desconto.
- Valor Líquido.

## Localização

Quando necessário, informe os dados da obra ou do imóvel.

**Dados da Obra**

- Origem do Endereço.
- Logradouro.
- Número.
- Complemento.
- Bairro.
- CEP.
- Município.
- Número da ART.
- Número do CNO/CEI.
- Número da Matrícula.
- Número do Projeto.

**Imóvel**

- Inscrição Imobiliária Fiscal.
- Código de Cadastro Imobiliário Brasileiro.
- Logradouro.
- Número.
- Complemento.
- Bairro.
- CEP.

## Demais

Nesta etapa são informadas informações complementares específicas da NFSe.

Conforme a operação, poderão ser preenchidas as abas:

- Substituição.
- Intermediário.
- Compra Governamental.
- Exploração Rodoviária.
- Atividade do Evento.
- Locação/Sub-Locação.
- Exportação.
- Total de Tributação.

## Informações Adicionais

Quando necessário, informe:

- Informações Adicionais de Interesse do Fisco.
- Informações Adicionais de Uso Interno.
- Tags.

**Finalizando**

Após revisar todas as informações, clique em **Salvar** para emitir a NFSe/DPS.
