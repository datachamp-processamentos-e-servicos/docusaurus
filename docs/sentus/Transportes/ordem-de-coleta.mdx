## Cadastrar uma Ordem de Coleta

Acesse:

**Transporte → Ordem de Coleta**

1. Clique em **+ Novo** para criar uma nova ordem de coleta ou selecione uma existente na lista para editá-la.
2. Preencha as informações da coleta.
3. Informe os participantes e os documentos relacionados.
4. Clique em **Salvar** para concluir o cadastro.

## Dados da Coleta

Nesta seção são registradas as informações principais da ordem de coleta.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

Principais informações:

- Data de emissão.
- Emitente.

## Participantes

Informe os participantes envolvidos na coleta.

Os principais campos são:

- Remetente.
- Destinatário.

## Documentos

Adicione os documentos relacionados à coleta.

Para cada documento informe:

- Quantidade.
- Unidade.
- Descrição.
- Documento.

Clique em **Add** para incluir o documento na ordem de coleta.

## Observações

Utilize este campo para registrar informações complementares sobre a coleta, quando necessário.

**Finalizando**

Após revisar todas as informações, clique em **Salvar** para concluir o cadastro da Ordem de Coleta.
