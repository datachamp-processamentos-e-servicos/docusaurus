## Cadastrar um Contrato de Transporte

Acesse:

**Transporte → Contratos**

1. Clique em **+ Novo** para criar um novo contrato ou selecione um contrato existente na lista para editá-lo.
2. Preencha as informações do contrato.
3. Informe os participantes, veículos e demais dados necessários.
4. Clique em **Salvar** para concluir o cadastro.

## Dados do Contrato

Nesta seção são registrados os dados principais do contrato.

Preencha as informações necessárias:

- Modelo.
- Série e Número.
- Data de emissão.
- Data de saída e chegada.
- Operação.
- CIOT, quando aplicável.
- Emitente.

## Participantes

Informe os participantes envolvidos no contrato.

Os principais campos são:

- Contratante.
- Contratado.
- Destinatário.
- Condutor.

## Valores

Informe os valores relacionados ao transporte.

Os principais campos são:

- Peso bruto.
- Valor do frete.
- Valor do adiantamento.
- Valor do pedágio.
- Valor adicional.
- Quantidade de entregas.

## Veículos

Adicione os veículos utilizados no contrato.

Para cada veículo informe:

- Veículo.
- Placa.
- RNTRC.
- Quantidade de eixos.

Clique em **Add** para incluir o veículo.

## Origem / Destino

Cadastre os locais atendidos pelo contrato.

Informe:

- Origem.
- Destino.
- Distância.
- Observação.

## Pagamentos

Caso existam dados bancários para pagamento, informe:

- Tipo.
- Agência.
- Conta.
- PIX.
- Recebedor.

## Dados da Carga

Nesta seção são informadas as características da carga transportada.

Informe, quando necessário:

- Natureza da carga.
- Tipo de carga.
- Indicador de desempenho.
- Indicador de retorno vazio.
- Composição veicular.
- Contratantes da carga fracionada.

## Observações

Utilize este campo para registrar informações complementares relacionadas ao contrato.

## Documentos

Associe os documentos vinculados ao transporte.

As principais informações são:

- Local de carga.
- Local de descarga.
- Documento.
- Peso bruto.

Clique em **Add** para adicionar cada documento.

**Finalizando**

Após revisar todas as informações, clique em **Salvar** para concluir o cadastro do Contrato de Transporte.
