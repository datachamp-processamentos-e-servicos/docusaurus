## Emitir um Relatório

Acesse:

**Transporte → Relatórios**

1. Selecione o relatório desejado na lista.
2. Clique no ícone de **Configurações** correspondente ao relatório.
3. Informe os filtros necessários.
4. Clique em **Visualizar** para gerar o relatório.

## Relatórios Disponíveis

Os principais relatórios do módulo são:

- Relatório de CT-e.
- Relatório de CT-e OS.
- Relatório de CT-e OS - Serviços.
- Relatório de CT-e por Forma de Pagamento.
- Relatório de MDF-e.
- Relatório de OS de Transporte.

## Filtros

Dependendo do relatório escolhido, podem ser informados filtros como:

- Emitente.
- Período (Data Inicial e Final).
- Tipo de emissão.
- Emissor.
- Remetente.
- Destinatário.
- Condutor.
- Situação.
- Placa.

Quanto mais filtros forem informados, mais específico será o resultado.

## Visualização

Após preencher os filtros desejados, clique em **Visualizar** para gerar o relatório com as informações encontradas.

**Finalizando**

Caso necessário, altere os filtros e gere o relatório novamente até obter o resultado desejado.
