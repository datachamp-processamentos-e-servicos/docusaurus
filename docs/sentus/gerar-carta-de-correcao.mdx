# Como gerar carta de correção
- Acesse Comercial → NFE;
- Localizar Nota Fiscal, para isso clique no ícone da lupa localizado no canto superior direito para realizar a busca;
- Em seguida, na linha correspondente à nota fiscal desejada, clique no botão Ações (+) eventos, e selecione a opção **Carta de Correção**;
- No campo Correção, informe as alterações que precisam ser realizadas e, após preencher as informações, clique em **Enviar**.\

:::note[Nota]
Observações Importantes
•	A descrição da correção deve ser clara e objetiva.
•	O campo de correção exige no mínimo 15 caracteres.
•	A Carta de Correção não pode ser utilizada para alterar informações que impactem:
o	Valores fiscais;
o	Base de cálculo dos impostos;
o	Quantidade ou valor dos produtos;
o	Dados do remetente ou destinatário que impliquem mudança de identificação;
o	Data de emissão da nota fiscal.

:::
