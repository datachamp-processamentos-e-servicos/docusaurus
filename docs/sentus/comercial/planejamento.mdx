# Planejamento Ⓧ

:::note[Antes de começar]

- O Planejamento é utilizado para definir metas comerciais por período, permitindo associar valores, representantes e outras informações para acompanhamento.
- Ao posicionar o cursor do mouse sobre um campo, o sistema exibe uma breve explicação sobre o preenchimento.

## Cadastrar um Planejamento

Acesse:

**Comercial → Auxiliares → Planejamento**

1. Clique em **+ Novo** para criar um planejamento.
2. Informe os dados principais.
3. Clique em **Salvar**.

---

## Informações do Planejamento

Nesta seção são definidos os dados utilizados para o planejamento comercial.

Preencha as informações mais importantes:

- **Emissão**, indicando o período do planejamento.
- **Valor** da meta ou planejamento.
- **Situação** (Ativo ou Inativo).
- **Emitente** responsável.
- **Representante**, quando o planejamento for específico.
- **Tags**, para facilitar filtros e organização.
- **Observação**, caso seja necessário registrar informações adicionais.

---

## Finalizando

Após conferir as informações preenchidas, clique em **Salvar** para concluir o cadastro do planejamento.
