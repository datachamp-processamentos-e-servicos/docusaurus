Auxiliares Tabela de Preço

Visão Geral​

A tela de Tabela de Preço é utilizada para definir regras de precificação dos produtos comercializados pela empresa.

Por meio desta funcionalidade, é possível criar diferentes tabelas de preço para cenários específicos, como vendas à vista, clientes especiais, representantes ou campanhas comerciais.

As tabelas de preço também permitem configurar margens de lucro e controlar quais valores serão utilizados durante as negociações e vendas.

Observações Importantes​

  • Todos os campos marcados com (*) são obrigatórios.
  • A margem configurada influencia diretamente a formação dos preços.
  • Mantenha as tabelas atualizadas para evitar divergências comerciais.
  • Utilize nomes claros para facilitar a identificação das tabelas.
  • Antes de substituir uma tabela, revise os impactos nos processos de venda.
  • A situação da tabela determina sua disponibilidade para utilização.

Principais Funcionalidades​

Novo Cadastro​

Clique em + Novo para criar uma nova tabela de preço.

Preencha as informações obrigatórias e configure os parâmetros de precificação desejados.

Após concluir o preenchimento, clique em Salvar para registrar a tabela.

Informações Cadastrais​

Nesta seção são definidos os dados principais da tebela de preço.

Nome​

Informe um nome que falicite a identificação da tabela.

Exemplos:

  • Tabela Base
  • Tabela À Vista
  • Tabeka Funcionários
  • Tabela Promocional

Situação​

Define a disponibilidade de tabela de preço.

As opções podem incluir:

  • Ativo
  • Restrição
  • Bloqueado
  • Removido
  • Inativo

Somente tabelas ativas poderão ser utilizadas normalmente nas operações.

Tipo Tabela​

Selecione o tipo de tabela de preço que será utilizada.

Esta configuração determina a forma como os valores serão calculados ou aplicados.

Margem​

Informe o percentual de margem de lucro que será aplicado sobre os produtos.

Esta margem é utilizada nos cálculos de markup e formação de preço.

Observação​

Utilize este campo para registrar informações complementares sobre a tabela de preço.

Demais Funcionalidades​

Controle de Precificação​

As tabelas de preço permitem criar diferentes estratégias comerciais para cada cenário de venda.

Isso possibilita manter preços específicos para determinados clientes, operações ou campanhas promocionais.

Menu​

Visão Geral​

Permite consultar informações relacionadas às tabelas de preço vinculadas aos clientes.

Utilize esta opção para visualizar rapidamente as configurações aplicadas às operações comerciais.

Substituir Tabela de Preço​

Permite atualizar uma tabela existente substituindo seus valores por uma nova configuração.

Esta funcionalidade é útil quando há necessidade de reajustar preços sem perder o histórico da tabela anterior.

Nova Tabela de Preço​

Durante o precesso de substituição, selecione a tabela antiga e clique em Avançar.

Em seguida, crie uma nova tabela e configure os parâmetros desejados.

O sistema utilizará a nova configuração conforme as regras definidas.

Pesquisa Avançada​

Clique em Pesquisa Avançada para localizar tabelas de preço específicas.

Informe um ou mais filtros e clique em Pesquisar.

Filtros Disponíveis​

Nome​

Localiza tabelas através do nome cadastrado.

Tipo​

Permite localizar tabelas através de informações registradas no campo de observação.

Situação​

Permite pesquisar tabelas conforme sua situação cadastral.

Abra o cadastro e realize as alterações necessárias.

Após concluir as modificações, clique em Salvar.

Dúvidas Frequentes​

O que é uma Tabela de Preço?​

É uma configuração utilizada para controlar os preços praticados pela empresa em diferentes cenários comerciais.

Como criar uma nova Tabela de Preço?​

Clique em + Novo.

Preencha os dados da tabela, informe a margem desejada e clique em Salvar.

Para que serve a margem da tabela?​

A margem define o percentual utilizado na formação dos preços e no cálculo do markup dos produtos.

Posso criar tabelas diferentes para tipos de clientes?​

Sim.

É possível criar tabelas específicas para diferentes estratégias comerciais, como clientes especiais, vendas à vista ou campanhas promocionais.

Como atualizar os preços de uma tabela existente?​

Acesse Menu > Substituir Tabela de Preço.

Selecione a tabela atual e crie uma nova versão com os valores desejados.

Como localizar uma tabela cadastrada?​

Utilize a Pesquisa Avançada e informe filtros como nome, tipo ou situação.

O que acontece se uma tabela estiver inativa?​

A tabela não poderá ser utilizada normalmente nas operações comerciais até que seja reativada.
