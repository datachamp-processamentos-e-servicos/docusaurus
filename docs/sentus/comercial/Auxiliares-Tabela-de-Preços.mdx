# Auxiliares Tabela de Preço Ⓧ

:::note[Antes de começar]

- As Tabelas de Preço são utilizadas para definir os valores dos produtos durante as operações comerciais.
- Ao posicionar o cursor do mouse sobre um campo, o sistema exibe uma breve explicação sobre o preenchimento.

## Cadastrar uma Tabela de Preço

Acesse:

**Comercial → Auxiliares → Tabelas de Preço**

1. Clique em **+ Novo** para criar uma nova tabela.
2. Informe os dados da tabela.
3. Clique em **Salvar**.

---

## Informações da Tabela

Nesta seção são configuradas as informações principais da tabela de preços.

Preencha os campos mais importantes:

- **Nome** da tabela.
- **Situação** (Ativo, restrição, bloqueado, removido ou Inativo).
- **% Margem**, quando utilizada.
- **Tipo da Tabela**, definindo como os preços serão calculados.
- **Observação**, se necessário.

---

## Tipos de Tabela

O sistema permite diferentes formas de cálculo para a tabela de preços.

Os tipos disponíveis podem variar conforme a configuração da empresa, sendo utilizados para definir como os valores dos produtos serão obtidos.

---

**Finalizando**

Após revisar as informações, clique em **Salvar** para concluir o cadastro da tabela de preços.
