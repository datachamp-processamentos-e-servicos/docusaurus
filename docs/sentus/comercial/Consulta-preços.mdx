## Consultar Preços

Acesse:

**Comercial → Consulta de Preços**

1. Selecione a **Data** e a **Tabela de Preços** que deseja consultar.
2. Localize o item utilizando a **Referência**, **Código de Barras**, **Nome**, **Código** ou **EAN**.
3. Consulte as informações apresentadas na lista de itens.

## Informações da Consulta

Nesta seção são exibidas as informações dos itens encontrados conforme os filtros informados.

A lista apresenta informações como:

- Código;
- Nome;
- Unidade;
- Valor Unitário.

Essas informações permitem consultar rapidamente os preços praticados para os itens cadastrados na tabela de preços selecionada.

---

**Finalizando**

Após localizar o item desejado, utilize as informações apresentadas para consulta dos preços cadastrados.
