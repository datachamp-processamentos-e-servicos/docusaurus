# Cotação  Ⓧ

## Visão Geral

A tela de Cotações é utilizada para elaborar propostas comerciais antes da efetivação da venda.

Nela é possível selecionar clientes, informar produtos, definir preços, descontos, condições de pagamento e dados de transporte, permitindo a negociação antes da geração do pedido.

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para criar uma nova cotação.

Preencha os campos obrigatórios e clique em **Salvar** para concluir o cadastro.

### Informações da Cotação

#### Número

Identificador gerado automaticamente pelo sistema.

#### Emissão

Data e hora da criação da cotação.

#### Embarque

Data prevista para embarque dos produtos.

#### Ordem de Compra

Número da ordem de compra vinculada à negociação.

#### Operação

Define o tipo de operação utilizada na cotação.

#### Situação

Apresenta o status atual da cotação.

### Cliente

Permite selecionar o cliente que receberá a proposta.

A pesquisa pode ser realizada por:

- CPF/CNPJ
- Nome
- Código

Também são exibidas informações como:

- Nome
- Documento
- Município
- Telefone

### Informações Comerciais

#### Representante

Responsável pela negociação.

#### Modalidade de Frete

Define como será realizado o transporte.

#### Forma de Pagamento

Condição de pagamento negociada.

#### Operação Tributária

Define a tributação aplicada à venda.

#### Prioridade

Nível de prioridade da negociação.

#### Emitente

Empresa responsável pela emissão dos documentos fiscais.

### Itens

Permite incluir os produtos ou serviços da cotação.

Os principais campos são:

- Referência / EAN
- Código ou Nome
- Unidade
- Quantidade
- Valor Unitário
- Valor da Mercadoria
- Desconto
- Valor de Tabela
- Pedido de Compra

### Transporte

Permite informar os dados de transporte da cotação.

#### Responsável pelo Transporte

Opções disponíveis:

- A Definir
- Emitente
- Cliente
- Transportador

#### Transportador

Permite informar:

- Nome
- Documento
- Inscrição Estadual
- E-mail
- Telefone
- Endereço
- Município
- CEP

## Mais Ações

### Editar

Permite alterar os dados do item informado.

### Excluir

Remove o item da cotação.

### Mais Informações

Exibe informações adicionais relacionadas aos impostos do item.

## Dúvidas Frequentes

### Como criar uma nova cotação?

Clique em **+ Novo**, preencha os dados da negociação e clique em **Salvar**.

### Como adicionar produtos?

Acesse a seção de itens, informe os produtos desejados e confirme a inclusão.

### Como informar descontos?

Utilize os campos de desconto disponíveis no item da cotação.

### Como selecionar um cliente?

Pesquise pelo CPF/CNPJ, nome ou código do cliente.

### Posso alterar o CPF/CNPJ após o cadastro?

Não é recomendado, pois o sistema poderá interpretar a alteração como um novo registro.

## Observações Importantes

- Todos os campos marcados com (*) são obrigatórios.
- Revise os valores antes de concluir a cotação.
- Verifique as condições de pagamento e tributação antes de salvar.
- Confirme os dados do cliente antes da emissão da proposta.
