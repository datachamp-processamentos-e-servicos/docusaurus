# Ponto de Vendas Ⓧ

:::note[Antes de começar]

- O Ponto de Venda (PDV) permite realizar vendas rápidas, adicionar produtos, registrar pagamentos e emitir NFC-e ou Pedido.
- Ao posicionar o cursor do mouse sobre um campo, o sistema exibe uma breve explicação sobre o preenchimento.

## Realizar uma Venda

Acesse:

**Comercial → Ponto de Venda**

1. Pesquise o produto pela **Referência**, **Código de Barras** ou **Nome**.
2. Informe a **Quantidade**.
3. Confirme o item para adicioná-lo à venda.
4. Repita o processo para incluir outros produtos.

---

## Lista de Itens

Nesta seção são exibidos todos os produtos adicionados à venda.

É possível visualizar:

- Referência.
- Descrição.
- Unidade.
- Quantidade.
- Valor Unitário.
- Total do item.
- Subtotal da venda.
- Valor de desconto.
- Valor do frete.
- Indicador de Presença.
- Total geral.

Também é possível editar ou remover itens antes da finalização.

---

## Pagamento

Após incluir os produtos, informe a forma de pagamento.

Configure:

- **Portador**.
- **Forma de Pagamento**.
- **Valor Recebido**.
- **Troco**, quando houver.

Utilize o botão **+** para adicionar uma ou mais formas de pagamento na mesma venda.

---

## Detalhes da Venda

Caso necessário, informe:

- **Participante (Cliente)**.
- **Representante**.
- **E-mail**.
- **Observações**.

A operação utilizada será exibida automaticamente conforme a configuração da venda.

---

## Finalizando

Após conferir todas as informações, utilize uma das opções disponíveis:

- **Salvar Pedido** para registrar a venda como pedido.
- **Salvar NFC-e** para emitir a Nota Fiscal de Consumidor Eletrônica.
- **+ Novo** para iniciar uma nova venda.
Após revisar as informações, clique em **Salvar NFC-e** para emitir a nota fiscal ou em **Salvar Pedido** para concluir a venda.
