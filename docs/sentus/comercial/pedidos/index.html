Pedidos Ⓧ

Visão Geral​

A tela de Pedidos é utilizada para registrar vendas realizadas por representantes ou através do sistema.

Por meio desta funcionalidade é possível controlar pedidos, acompanhar faturamentos, emitir documentos, realizar conferências e acompanhar todo o processo comercial.

Observações Importantes​

  • Todos os campos obrigatórios devem ser preenchidos.
  • Revise as informações antes de faturar o pedido.
  • Verifique a situação do pedido antes da emissão fiscal.
  • Utilize a conferência para validar os produtos antes da entrega.
  • O faturamento gera os documentos fiscais relacionados à venda.

Principais Funcionalidades​

Novo Cadastro​

Clique em + Novo para criar um novo pedido.

Preencha os campos obrigatórios e clique em Salvar para concluir o cadastro.

Informações do Pedido​

Número​

Código gerado automaticamente pelo sistema.

Emissão​

Data e hora da emissão do pedido.

Embarque​

Data prevista para envio dos produtos.

Cliente​

Cliente responsável pela compra.

Prioridade​

Nível de prioridade da venda.

Situação​

Status atual do pedido.

Observações​

Informações complementares relacionadas ao pedido.

Dados Financeiros​

Informações financeiras vinculadas à venda.

Menu​

Checkout (Faturar / Encerrar)​

Realiza o faturamento do pedido e a emissão dos documentos fiscais.

Totalizar​

Exibe os valores totais e quantidades dos pedidos selecionados.

Etiqueta​

Gera etiquetas para transporte ou identificação de volumes.

Exportador​

Permite gerar documentos para impressão ou compartilhamento.

Enviar E-mail​

Envia as informações do pedido por e-mail.

Desmembrar​

Divide um pedido em dois ou mais pedidos.

Exportar​

Exporta os dados exibidos para planilha.

Integração E-commerce​

Permite integração com plataformas de comércio eletrônico quando configuradas.

Exportar​

Acesse Menu > Exportador.

Os formatos disponíveis podem incluir:

  • PDF
  • XML
  • Impressão Térmica

Também é possível definir:

  • Modelo de impressão
  • Layout
  • Ficha técnica

Pesquisa Avançada​

Utilize a Pesquisa Avançada para localizar pedidos específicos.

Filtros Disponíveis​

  • Número
  • Ordem de Compra
  • Cliente
  • Representante
  • Operação
  • Pagamento
  • Origem
  • Frete
  • Situação

Filtros Adicionais​

  • Somente Pendentes
  • Sem Transportador

Filtro por Data​

Permite consultar registros por:

  • Emissão
  • Embarque
  • Inclusão

Mais Ações​

Imprimir​

Imprime as informações do pedido.

Exportador​

Permite exportar o pedido em PDF ou XML.

Situação​

Exibe a situação atual do pedido.

Faturar​

Realiza o faturamento da venda.

Visão Geral​

Exibe informações resumidas do pedido e dos itens faturados.

Conferência​

Permite validar os produtos antes da expedição ou faturamento.

Duplicar​

Cria um novo pedido utilizando os mesmos dados do pedido selecionado.

Arquivos​

Permite anexar documentos ao pedido.

Download​

Realiza o download dos arquivos anexados.

Dúvidas Frequentes​

Como criar um novo pedido?​

Clique em + Novo, preencha os dados obrigatórios e clique em Salvar.

Como faturar um pedido?​

Utilize a opção Checkout (Faturar / Encerrar) ou Mais Ações > Faturar.

Como localizar um pedido?​

Utilize a Pesquisa Avançada e informe os filtros desejados.

Como exportar um pedido?​

Acesse Menu > Exportador ou Mais Ações > Exportador.

Como duplicar um pedido?​

Selecione o pedido desejado e utilize Mais Ações > Duplicar.

Como enviar um pedido por e-mail?​

Acesse Menu > Enviar E-mail e informe o destinatário.
