Pedidos

Visão Geral​

A tela de Pedidos é utilizada para registrar e gerenciar pedidos de venda realizados por representantes ou através da plataforma Sentus.

Nela é possível selecionar clientes, adicionar produtos, configurar condições comerciais, definir transporte e acompanhar o faturamento do pedido.

Observações Importantes​

  • Os campos marcados com (*) são obrigatórios.
  • O número do pedido é gerado automaticamente pelo sistema.
  • Algumas funcionalidades são restritas ao administrador do ERP.
  • O faturamento gera os documentos fiscais relacionados ao pedido.

Principais Funcionalidades​

Novo Cadastro​

Clique em + Novo para criar um pedido.

Preencha as informações do cliente, dados comerciais, itens e transporte.

Após concluir o preenchimento, clique em Salvar.

Informações Cadastrais​

Cabeçalho​

Permite informar os dados principais do pedido:

  • Número
  • Emissão
  • Embarque
  • Ordem de Compra
  • Operação
  • Situação

Cliente​

Permite localizar e vincular o cliente ao pedido através de:

  • Nome
  • Código
  • CPF/CNPJ

Informações Comerciais​

Permite configurar:

  • Representante
  • Forma de Pagamento
  • Modalidade de Frete
  • Operação Tributária
  • Prioridade
  • Emitente

Itens​

Adicione os produtos que farão parte do pedido.

Para cada item é possível informar:

  • Produto
  • Quantidade
  • Valor Unitário
  • Desconto

O sistema calcula automaticamente os valores totais do pedido.

Transporte​

Permite informar:

  • Responsável pelo transporte
  • Transportadora
  • Dados de contato
  • Endereço de entrega

Menu​

Checkout Faturar / Encerrar​

Realiza o faturamento ou encerramento do pedido.

Totalizar​

Exibe os totais dos pedidos cadastrados.

Etiqueta​

Gera etiquetas de transporte.

Exportador​

Permite gerar arquivos para impressão ou compartilhamento.

Enviar E-mail​

Envia o pedido por e-mail.

Desmembrar​

Divide um pedido em mais de um pedido.

Integração E-commerce​

Disponível para empresas que possuem integração com plataformas de e-commerce.

Pesquisa Avançada​

Utilize a Pesquisa Avançada para localizar pedidos específicos.

Filtros Disponíveis​

  • Número
  • Ordem de Compra
  • Cliente
  • Representante
  • Operação
  • Forma de Pagamento
  • Origem
  • Frete
  • Situação
  • Somente Pendentes
  • Sem Transportador
  • Período

Mais Ações​

Editar​

Permite alterar as informações do pedido.

Excluir​

Permite remover itens do pedido.

Imprimir​

Imprime os dados do pedido.

Exportador​

Exporta o pedido em PDF ou XML.

Faturar​

Realiza o faturamento do pedido.

Visão Geral​

Exibe um resumo das informações do pedido.

Conferência​

Encaminha o pedido para conferência.

Duplicar​

Cria um novo pedido com base em um pedido existente.

Arquivos​

Permite anexar documentos ao pedido.

Download​

Permite baixar arquivos anexados.

Dúvidas Frequentes​

Como criar um pedido?​

Clique em + Novo, preencha os dados necessários e clique em Salvar.

Como adicionar produtos?​

Acesse a seção Itens, selecione os produtos e confirme a inclusão.

Como faturar um pedido?​

Acesse Menu > Checkout Faturar / Encerrar.

Como localizar um pedido?​

Utilize a Pesquisa Avançada e informe os filtros desejados.

Observações Importantes​

  • Os campos marcados com (*) são obrigatórios.
  • O número do pedido é gerado automaticamente pelo sistema.
  • Algumas funcionalidades são restritas ao administrador do ERP.
  • O faturamento gera os documentos fiscais relacionados ao pedido.
