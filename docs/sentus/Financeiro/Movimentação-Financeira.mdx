## Consultar Movimentações

Acesse:

**Financeiro → Movimentação**

1. Selecione o **Portador** que deseja consultar;
2. Informe o período **De** e **Até**;
3. Caso necessário, informe uma data de **Fechamento**;
4. Clique em **Pesquisar**.

## Cadastrar Nova Movimentação

1. Clique em **+ Novo** para cadastrar uma nova movimentação, ou clique em alguma da lista abaixo para editá-la.

## Informações da Movimentação

Nesta seção são registrados os dados principais da movimentação.

- Categoria;
- Tipo;
- Emitente;
- Portador;
- Grupo Financeiro;
- Participante;
- Tags (opcional);
- Observação (opcional).

O campo **Situação** permite selecionar o status : Ativo, restrição, bloqueado, removido ou inativo.

---

**Finalizando**

Após revisar todas as informações, clique em **Salvar** para concluir o cadastro da movimentação.
