# Lançamentos Financeiros Ⓧ

## Visão Geral

A tela de Lançamentos Financeiros é utilizada para registrar entradas e saídas financeiras realizadas pela empresa.

Por meio desta funcionalidade, é possível controlar movimentações financeiras, organizar lançamentos por categoria, acompanhar valores e manter o histórico das operações realizadas.

---

## Observações Importantes

- Todos os campos marcados com (*) são obrigatórios.
- O CPF/CNPJ não deve ser alterado em um cadastro já existente.
- Os lançamentos podem representar entradas ou saídas financeiras.
- A contabilização está disponível apenas para usuários administradores.
- Utilize corretamente os centros de custo e grupos financeiros para manter os relatórios organizados.
- Revise os dados antes de concluir o cadastro.

---

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para criar um novo lançamento financeiro.

Preencha os campos obrigatórios marcados com (*) e clique em **Salvar**.

### Informações Cadastrais

Nesta seção são informados os dados principais do lançamento.

#### Categoria

Define a categoria finacneira do lançamento.

Exemplos:

- Caixa
- Conta Corrente
- Outras categorias cadastradas

#### Tipo

Define se o lançamento será:

- Entrada
- Saída

#### Emitente

Selecione o emitente responsável pelo lançamento.

#### Situação

Define a situação atual do registro.

Exemplos:

- Ativo
- Inativo

#### Data

Informe a data e hora do lançamento.

#### Valor

Informe o valor da movimentação financeira.

#### Centro de Custo

Selecione o centro de custos relacionado ao lançamento.

#### Grupo Financeiro

Define o grupo financeiro ao qual o lançamento pertence.

### Participante

Permite vincular um cliente, fornecedor ou participante ao lançamento.

Os principais campos disponíveis são:

#### Pesquisa por CPF/CNPJ, Nome ou Código

Permite localizar participantes já cadastrados.

#### CPF/CNPJ

Informe o documento do participante.

#### Tags

Permite localizar ou classificar participantes através de etiquetas.

#### Observação

Utilize para registrar informações complementares relacionadas ao participante.

## Menu

### Contabilização

Permite realizar processos de contabilização relacionados aos lançamentos.

Esta funcionalidade está disponível apenas para usuários administradores.

### Exportar

Exporta os registros exibidos para uma planilha Excel.

Após a geração, o download será realizado automaticamente.

### Imprimir

Gera uma impressão dos registros exibidos na tela.

## Pesquisa Avançada

Utilize a pesquisa Avançada para localizar lançamentos específicos.

Informe os filtros desejados e clique em **Pesquisar**.

## Filtro Disponíveis

- Emitente
- CPF/CNPJ
- Nome
- Tipo
- Categoria
- Origem
- Situação

### Filtro por Data

Permite consultar lançamentos dentro de um período específico.

O filtro pode ser aplicado utilizando a data de emissão do lançamento.

## Mais Ações

### Recibo Compacto

## Dúvidas Frequentes

### Como cadastrar um lançamento financeiro?

Clique em **+ Novo**, preencha os dados obrigatórios e clique em **Salvar**.

### Como localizar um lançamento?

Utilize a **Pesquisa Avançada** e informe os filtros desejados.

### Como gerar um recibo?

Localize o lançamento desejado e acesse **Mais Ações > Recibo**.

### Como exportar os lançamentos?

Acesse **Menu > Exportar** para gerar uma planilha Excel.

### Como anexar documentos ao lançamento?

Acesse **Mais Ações > Arquivos** e realize o envio do documento desejado.

### Quem pode realizar contabilizações?

Apenas usuários com permissões administrativas.
