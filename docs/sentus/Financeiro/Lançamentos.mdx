# Adiantamento Lançamentos Ⓧ

:::note[Antes de começar]

- Os lançamentos permitem registrar movimentações relacionadas aos adiantamentos cadastrados.
- Essas movimentações mantêm o controle financeiro e contábil dos valores adiantados.

:::

## Como acessar

Acesse:

**Financeiro → Adiantamentos → Lançamentos**

1. Clique em **Novo** para criar um lançamento ou abra um registro existente.
2. Preencha as informações necessárias.
3. Clique em **Salvar** para concluir o lançamento.

---

## Informações do Lançamento

Nesta seção são informados os principais dados do lançamento.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

Entre as informações disponíveis estão:

- Beneficiário;
- Participante;
- Número;
- Data de emissão;
- Valor;
- Tipo;
- Portador;
- Recurso financeiro;
- Centro de custo;
- Operação contábil;
- Observações.

---

## Finalizando

Após revisar as informações, clique em **Salvar** para concluir o lançamento.
