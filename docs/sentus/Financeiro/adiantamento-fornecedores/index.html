Adiantamentos de Fornecedores

Visão Geral​

A tela de Adiantamentos de Fornecedores é utilizada para registrar valores antecipados relacionados a fornecedores.

A funcionalidade permite controlar pagamentos antecipados, acompanhar liquidações e manter o histórico financeiro dos registros.

Observações Importantes​

  • Todos os campos obrigatórios devem ser preenchidos.
  • O CPF/CNPJ não deve ser alterado em um cadastro existente.
  • Algumas funcionalidades dependem das permissões do usuário.

Principais Funcionalidades​

Novo Cadastro​

Clique em + Novo para cadastrar um novo adiantamento.

Preencha os campos obrigatórios e clique em Salvar.

Informações Cadastrais​

Beneficiário​

Selecione o beneficiário do adiantamento.

Participante​

Selecione o fornecedor relacionado ao adiantamento.

Situação​

Define a situação do adiantamento.

Número​

Número de identificação do registro.

Emissão​

Data de emissão do adiantamento.

Valor​

Valor total do adiantamento.

Portador​

Selecione o portador responsável.

Recurso Financeiro​

Define a origem financeira utilizada na operação.

Operação Contábil​

Selecione a operação contábil correspondente.

Observações​

Utilize para registrar informações complementares.

Informações Adicionais​

Permite informar dados contábeis adicionais.

Menu​

Contabilização​

Disponível apenas para usuários administradores.

Exportar​

Exporta os registros exibidos para uma planilha Excel.

Pesquisa Avançada​

Utilize a Pesquisa Avançada para localizar registros específicos.

Filtros Disponíveis​

  • Beneficiário
  • Participante
  • Portador
  • Tipo
  • Situação
  • Data de Emissão

Mais Ações​

Editar​

Permite alterar informações previamente cadastradas.

Eventos​

Exibe movimentações vinculadas ao fornecedor.

Arquivos​

Permite consultar documentos anexados ao registro.

Anexar​

Disponível apenas para usuários administradores.

Permite anexar documentos relacionados ao adiantamento.

Dúvidas Frequentes​

Como cadastrar um adiantamento para fornecedor?​

Clique em + Novo, preencha os dados necessários e clique em Salvar.

Como localizar um registro?​

Utilize a Pesquisa Avançada e informe os filtros desejados.

Como anexar documentos?​

Acesse Mais Ações > Anexar.

Quem pode realizar contabilizações?​

Somente usuários administradores.
