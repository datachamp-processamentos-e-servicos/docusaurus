## Gerar Relatório

Acesse:

**Financeiro → Relatórios**

1. Clique no relatório desejado na lista para abrir os filtros.

## Filtros do Relatório

Nesta seção informe apenas os dados necessários para a geração do relatório.

- Emitente;
- Período (Data Inicial e Data Final);
- Tipo;
- Listagem.

Após definir os filtros, clique em **Visualizar** para gerar o relatório.

---

**Finalizando**

Confira as informações exibidas e utilize as opções disponíveis para imprimir ou exportar o relatório, se necessário.
