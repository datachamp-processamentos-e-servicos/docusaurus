## Cadastrar Novo Lançamento Financeiro

Acesse:

**Financeiro → Lançamentos**

1. Clique em **+ Novo** para cadastrar um lançamento financeiro, ou clique em algum da lista abaixo para editá-lo.

## Informações do Lançamento

Nesta seção são informados os dados principais do lançamento financeiro.

Informe apenas os campos necessários, como:

- Categoria;
- Tipo;
- Data;
- Valor;
- Portador;
- Grupo Financeiro.

## Participante

Caso o lançamento esteja vinculado a um participante, informe-o nesta seção.

É possível localizar o participante pesquisando pelo:

- Nome;
- CPF/CNPJ;
- Código.

## Informações Complementares

Caso necessário, utilize os campos de:

- Tags;
- Observação.

---

**Finalizando**

Após revisar as informações, clique em **Salvar** para concluir o lançamento financeiro.
