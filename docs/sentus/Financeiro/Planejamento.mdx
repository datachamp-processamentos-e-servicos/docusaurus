# Planejamento Financeiro

## Visão Geral

A funcionalidade de Planejamento Financeiro permite acompanhar movimentações financeiras dentro de um período específico, auxiliando no controle de entradas, saídas e saldos da empresa.

Por meio desta consulta, é possível visualizar informações consolidadas e acompanhar a situação financeira com maior facilidade.

---

## Observações Importantes

- Os resultados apresentados dependem das movimentações registradas no sistema.
- Utilize filtros de período para facilitar a análise das informações.
- Algumas informações podem variar conforme as permissões do usuário.

---

## Principais Funcionalidades

### Consulta de Planejamento

Informe os filtros desejados para consultar as movimentações financeiras.

Após definir os parâmetros da pesquisa, execute a consulta para visualizar os resultados disponíveis.

### Informações Disponíveis

Os principais dados apresentados são:

- Beneficiário
- Data Inicial
- Data Final
- Valores movimentados
- Saldos apurados

## Menu

### Imprimir

Permite gerar uma impressão das informações exibidas na tela.

Utilize esta opção para emitir relatórios ou realizar conferências das movimentações consultadas. 

## Mais Ações

### Visualizar Informações

Permite expandir os dados exibidos na consulta.

Ao utilizar esta opção, o sistema apresenta informações adicionais relacionadas aos clientes, fornecedores e demais registros vinculados ao período consultado.

## Dúvidas Frequentes

### Como consultar um período específico?

Informe a Data Inicial e a Data Final desejadas e execute a pesquisa.

### Como visualizar mais informações da consulta?

Utilize a opção disponível em **Mais Ações** para expandir os detalhes do planejamento.

### Como imprimir os resultados?

Acesse **Menu > Imprimir** para gerar a impressão dos dados exibidos.
