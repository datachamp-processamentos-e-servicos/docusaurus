# Cheques Recebidos

:::note[Antes de começar]

- Utilize este cadastro para registrar os cheques recebidos de clientes ou terceiros.
- O controle permite acompanhar a situação e o pagamento de cada cheque.

:::

## Como acessar

Acesse:

**Financeiro → Cheques → Recebidos**

1. Clique em **Novo** para cadastrar um cheque ou abra um registro existente.
2. Preencha as informações necessárias.
3. Clique em **Salvar** para concluir o cadastro.

---

## Informações do Cheque

Nesta seção são informados os principais dados do cheque.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

Entre as informações disponíveis estão:

- Emitente;
- Cliente/Fornecedor;
- Portador;
- Número do cheque;
- Valor;
- Data de emissão e vencimento;
- Centro de custo;
- Grupo;
- Vinculação;
- Operação contábil;
- Situação;
- Observações.

---

## Informações do Pagamento

Nesta seção podem ser registrados os dados relacionados à liquidação do cheque.

Entre as informações disponíveis estão:

- Código de barras;
- Data de pagamento;
- Juros;
- Desconto.

---

## Finalizando

Após revisar as informações, clique em **Salvar** para concluir o cadastro do cheque.
