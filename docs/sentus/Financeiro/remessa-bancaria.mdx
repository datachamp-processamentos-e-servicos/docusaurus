# Remessas Bancárias Ⓧ
 
:::note[Antes de começar]

- Os arquivos de remessa são utilizados para enviar informações de cobrança ao banco.
- Eles são geradps a partir dos títulos cadastrados e configurados para o portador bancário correspondente.

:::

## Como acessar

Acesse

**Financeiro → Cobrança Bancária → Remessas**

1. Localize o arquivo desejado.
2.Abra o registro para consultar suas informações.

---

## Informações do Arquivo

Nesta seção são apresentadas as informações gerais da remessa, como data de geração, tipo do arquivo, portador bancário, nome do arquivo e observações.

## Conteúdo da Remessa

O sistema exibe o conteúdo completo do arquivo gerado, permitindo a conferência das informações que serão enviadas ao banco.

---

## Finalizando

Após conferir os dados da remessa, o arquivo estará disponível para utilização conforme o processo de envio adotado pela instituição bancária.
