# Remessas Bancárias Ⓧ
 
## Visão Geral

A tela de Remessas Bancárias é utilizada para gerar arquivos de remessa que serão enviados ao banco para registro de boletos.

Cada banco exige uma remessa própria, sendo necessário gerar um arquivo para cada portador utilizado.

---

## Observações Importantes

- Cada banco deve possuir uma remessa própria.
- Verifique os títulos selecionados antes de gerar o arquivo.
- Utilize os filtros de data para facilitar a localização dos boletos.
- Os arquivos podem ser exportados em PDF ou XLS.

---

## Principais Funcionalidades

### Gerar Remessa

Informe os dados necessários para geração da remessa bancária.

Após preencher os filtros desejados, clique em **Pesquisar** para localizar os títulos disponíveis.

Os boletos encontrados poderão ser selecionados para compor o arquivo de remessa.

### Informações da Remessa

#### Portador

Selecione o banco responsável pelos boletos.

Cada banco deve possuir sua própria remessa.

#### Finalidade

Define o objetivo da remessa.

As opções disponíveis podem incluir:

- Cobrança
- Pagamento
- Integração

#### Data Inicial e Final

Informe o período utilizado para localizar os boletos que serão enviados ao banco.

### Lista de Títulos

Após a pesquisa, o sistema exibirá os títulos encontrados.

As principais informações exibidas são:

- Número
- Cliente ou Fornecedor
- Emissão
- Vencimento
- Valor
- Situação

## Menu

### Exportador

Permite exportar as remessas selecionadas.

Os formatos disponíveis são:

- PDF
- XLS

Após selecionar o formato desejado, clique em **Exportar**.

## Pesquisa Avançada

Utilize a Pesquisa Avançada para localizar remessas específicas.

### Filtros Disponíveis

#### Nome do Arquivo

Localiza remessas através do nome do arquivo gerado.

#### Finalidade

Permite pesquisar remessas conforme sua finalidade.

#### Portador

Permite localizar remessas através do banco portador.

#### Filtro de Data

Permite pesquisar remessas dentro de um período específico.

## Mais Ações

### PDF

Gera um arquivo PDF contendo as informações da remessa.

### Download

Realiza o download da remessa selecionada.

### XML

Permite baixar o arquivo XML da cobrança bancária.

### Retornos

Permite acessar os retornos relacionados aos boletos registrados junto ao banco.

## Dúvidas Frequentes

### O que é uma remessa bancária?

É o arquivo enviado ao banco para registrar boletos e transmitir informações financeiras.

### Preciso gerar uma remessa para cada banco?

Sim. Cada banco deve possuir sua própria remessa.

### Como localizar uma remessa já gerada?

Utilize a Pesquisa Avançada e informe os filtros desejados.

### Como baixar uma remessa?

Acesse **Mais Ações > Download**.

### Como exportar remessas para Excel?

Acesse **Menu > Exportador** e selecione o formato XLS.
