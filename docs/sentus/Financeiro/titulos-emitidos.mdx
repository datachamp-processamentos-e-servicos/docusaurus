# Cheques Emitidos

## Visão Geral

A tela de Cheques Emitidos é utilizada para registrar e controlar os cheques emitidos pela empresa.

Por meio desta funcionalidade, é possível acompanhar vencimentos, controlar liquidações, registrar informações financeiras e consultar a situação dos cheques emitidos.

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para cadastrar um novo cheque emitido.

Preencha os campos obrigatórios e informe os dados relacionados ao cheque.

Após concluir o preenchimento, clique em **Salvar**.

### Informações do Cheque

Nesta seção são registrados os dados principais do cheque.

#### Emitente

Selecione a empresa ou unidade responsável pela emissão do cheque.

#### Cliente/Fornecedor

Informe o favorecido que receberá o cheque.

#### Portador

Selecione o portador do cheque, como banco ou carteira.

#### Série

Informe a série do cheque.

#### Número

Informe o número do cheque.

#### Nominal

Informe o nome do favorecido quando o cheque for nominal.

#### Emissão

Data de emissão do cheque.

#### Vencimento

Data prevista para compensação ou vencimento.

#### Valor

Valor total do cheque.

#### Grupo

Grupo financeiro relacionado ao lançamento.

#### Operação Contábil

Define a operação contábil vinculada ao cheque.

#### Conta Contábil

Conta contábil utilizada no lançamento.

#### Situação

Permite controlar o status do cheque.

Exemplos:

- Em Aberto
- Liquidado

#### Observações

Utilize para registrar informações complementares.

### Informações do Pagamento

Permite registrar dados da liquidação do cheque.

#### Código de Barras

Informe o código de barras do envelope ou documento relacionado.

#### Data Pagamento

Data em que ocorreu a liquidação.

#### Juros

Valor dos juros aplicados.

#### Desconto

Valor dos descontos concedidos.

## Menu

### Contabilização

Permite realizar processos contábeis relacionados aos cheques.

Disponível apenas para usuários com permissões administrativas.

### Liquidação em Lote

Permite liquidar vários cheques simultaneamente.

Os principais dados informados são:

- Data de Pagamento
- Valor Total
- Desconto
- Juros
- Operação Contábil

### Totalizar

Exibe os valores totais dos cheques cadastrados.

Permite consultar rapidamente valores em aberto e liquidados.

### Imprimir

Gera um relatório contendo os cheques emitidos.

### Exportar

Exporta os registros exibidos para uma planilha Excel.

## Pesquisa Avançada

Utilize a Pesquisa Avançada para localizar cheques específicos.

### Filtros Disponíveis

- Emitente
- Número
- Nominal
- Participante
- Observações
- Portador
- Operação
- Grupo
- Origem
- Situação

### Ordenação

Os resultados podem ser ordenados por:

- Número
- Vencimento

### Filtro por Data

Permite consultar registros por:

- Emissão
- Vencimento
- Pagamento

## Dúvidas Frequentes

### Como cadastrar um cheque emitido?

Clique em **+ Novo**, preencha os dados obrigatórios e clique em **Salvar**.

### Como liquidar vários cheques ao mesmo tempo?

Acesse **Menu > Liquidação em Lote** e informe os dados da liquidação.

### Como localizar um cheque específico?

Utilize a **Pesquisa Avançada** e informe os filtros desejados.

### Como exportar os registros?

Acesse **Menu > Exportar** para gerar uma planilha Excel.

### Quem pode acessar a contabilização?

Apenas usuários com permissões administrativas.

## Observações Importantes

- Todos os campos marcados com (*) são obrigatórios.
- O número do cheque deve ser informado corretamente.
- A liquidação em lote agiliza o processamento de múltiplos registros.
- Algumas funcionalidades dependem das permissões do usuário.
- Revise os dados antes de concluir a liquidação.

## Objetivo da Tela

Controlar os cheques emitidos pela empresa, permitindo acompanhar vencimentos, pagamentos, liquidações e informações financeiras relacionadas.
