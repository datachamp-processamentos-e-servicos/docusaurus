# Adiantamento de Fornecedores

:::note[Antes de começar]

- Os adiantamentos permitem registrar valores pagos antecipadamente a fornecedores.
- Esses valores podem ser utilizados posteriormente para abatimento de títulos ou outras movimentações financeiras.

:::

## Como acessar

Acesse:

**Financeiro → Adiantamentos → Fornecedores**

1. Clique em **Novo** para criar um novo adiantamento ou abra um registro existente.
2. Preencha as informações necessárias.
3. Clique em **Salvar** para concluir o cadastro.

---

## Informações do Adiantamento

Nesta seção são informados os principais dados do adiantamento.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

Entre as informações disponíveis estão:

- Beneficiário;
- Participante;
- Número;
- Data de emissão;
- Valor;
- Tipo;
- Portador;
- Recurso financeiro;
- Centro de custo;
- Observações.

---

## Finalizando

Após revisar as informações, clique em **Salvar** para concluir o cadastro do adiantamento.
