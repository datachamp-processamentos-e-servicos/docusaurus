## Cadastrar Novo Adiantamento

Acesse:

**Financeiro → Adiantamentos → Fornecedores**

1. Clique em **+ Novo** para cadastrar um novo adiantamento, ou clique em algum da lista abaixo para editá-lo.

## Informações do Adiantamento

Nesta seção são registrados os dados principais do adiantamento.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

Entre as principais informações estão:

- Beneficiário;
- Participante;
- Número;
- Emissão;
- Valor;
- Tipo: define que o lançamento será um pagamento ao fornecedor;
- Portador;
- Recurso Financeiro: conta ou caixa utilizado para realizar o pagamento;
- Situação;
- Observações;
- Informações Adicionais.

---

**Finalizando**

Após revisar todas as informações, clique em **Salvar** para concluir o cadastro do adiantamento.
