# Cheques Recebidos

## Visão Geral

A tela de Cheques Recebidos é utilizada para registrar e controlar os cheques recebidos pela empresa.

A funcionalidade permite acompanhar recebimentos, controlar liquidações, registrar informações financeiras e consultar a situação de cada cheque.

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para cadastrar um novo cheque recebido.

Preencha os campos obrigatórios e clique em **Salvar**.

### Informações do Cheque

Nesta seção são registrados os dados principais do cheque recebido.

#### Emitente

Selecione o emitente relacionado ao cheque.

#### Cliente/Fornecedor

Informe o cliente ou participante responsável pelo cheque.

#### Portador

Selecione o portador do cheque, como banco ou carteira.

#### Série

Informe a série do cheque.

#### Número

Informe o número do cheque.

#### Nominal

Informe o favorecido quando o cheque for nominal.

#### Emissão

Data de emissão do cheque.

#### Vencimento

Data prevista para recebimento ou compensação.

#### Valor

Valor total do cheque.

#### Grupo

Grupo financeiro relacionado ao lançamento.

#### Operação Contábil

Operação contábil vinculada ao cheque.

#### Conta Contábil

Conta utilizada para contabilização.

#### Situação

Permite acompanhar o status do cheque.

Exemplos:

- Em Aberto
- Liquidado

#### Observações

Utilize para registrar informações adicionais.

### Informações do Pagamento

Permite controlar dados relacionados ao recebimento.

#### Código de Barras

Informe o código de barras relacionado ao documento.

#### Data Pagamento

Data em que o cheque foi recebido ou liquidado.

#### Juros

Valor dos juros recebidos.

#### Desconto

Valor dos descontos aplicados.

## Menu

### Contabilização

Permite executar processos de contabilização dos cheques.

Disponível apenas para usuários administradores.

### Liquidação em Lote

Permite liquidar diversos cheques recebidos simultaneamente.

Os principais campos são:

- Data de Pagamento
- Valor dos Cheques
- Valor Total
- Desconto
- Juros
- Operação Contábil

### Totalizar

Exibe os totais dos cheques cadastrados.

Permite consultar valores em aberto e valores liquidados.

### Imprimir

Gera um recibo ou relatório dos cheques recebidos.

### Exportar

Exporta os registros exibidos para uma planilha Excel.

## Pesquisa Avançada

Utilize a Pesquisa Avançada para localizar cheques recebidos.

### Filtros Disponíveis

- Emitente
- Número
- Nominal
- Participante
- Observações
- Portador
- Operação
- Grupo
- Origem
- Situação

### Ordenação

Os resultados podem ser organizados por:

- Número
- Vencimento

### Filtro por Data

Permite consultar registros por:

- Emissão
- Vencimento
- Pagamento

## Dúvidas Frequentes

### Como cadastrar um cheque recebido?

Clique em **+ Novo**, preencha as informações necessárias e clique em **Salvar**.

### Como consultar cheques em aberto?

Utilize a Pesquisa Avançada e filtre pela situação desejada.

### Como liquidar vários cheques ao mesmo tempo?

Acesse **Menu > Liquidação em Lote** e informe os dados da liquidação.

### Como imprimir um comprovante?

Acesse **Menu > Imprimir**.

### Como exportar os registros?

Acesse **Menu > Exportar** para gerar uma planilha Excel.

### Quem pode realizar contabilizações?

Apenas usuários com permissões administrativas.

## Observações Importantes

- Todos os campos obrigatórios devem ser preenchidos.
- Verifique os dados do cheque antes de salvar.
- A situação do cheque deve ser mantida atualizada.
- Algumas funcionalidades dependem das permissões do usuário.
- Utilize os filtros de pesquisa para localizar registros rapidamente.

## Objetivo da Tela

Controlar os cheques recebidos pela empresa, permitindo acompanhar recebimentos, liquidações, vencimentos e informações financeiras relacionadas aos lançamentos.
