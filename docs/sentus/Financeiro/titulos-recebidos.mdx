## Cadastrar Novo Título

Acesse:

**Financeiro → Títulos a Pagar**

1. Clique em **+ Novo** para cadastrar um novo título, ou clique em algum da lista abaixo para editá-lo.

## Informações do Título

Nesta seção são registrados os dados principais do título.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

Entre as principais informações estão:

- Cliente/Fornecedor;
- Beneficiário;
- Número;
- Parcela;
- Emissão;
- Vencimento;
- Valor;
- Centro de Custo: utilizado para classificar o lançamento financeiro;
- Grupo;
- Observações.

## Informações da Cobrança

Nesta seção podem ser informados os dados da cobrança do título.

Quando necessário, informe:

- Portador;
- Espécie do Título;
- Código de Barras;
- Número da Fatura.

## Informações do Pagamento

Nesta seção são registrados os dados do pagamento do título.

Informe, quando necessário:

- Data de Pagamento;
- Valor Pago;
- Situação;
- Conta Contábil: utilizada quando houver integração com a contabilidade.

---

**Finalizando**

Após revisar todas as informações, clique em **Salvar** para concluir o cadastro do título.
