## Cadastrar Novo Adiantamento

Acesse:

**Financeiro → Adiantamentos → Clientes**  

1. Clique em **+ Novo** para cadastrar um novo adiantamento, ou clique em algum da lista abaixo para editá-lo.

## Informações do Adiantamento

Nesta seção são registrados os dados principais do adiantamento, preencha os campos necessários.

Entre as principais informações estão:

- Beneficiário;
- Participante;
- Número;
- Emissão;
- Valor;
- Tipo: define se o lançamento será de recebimento ou pagamento;
- Portador;
- Recurso Financeiro: conta ou caixa utilizado para movimentar o valor;
- Situação;
- Observações;
- Informações Adicionais.

---

**Finalizando**

Após revisar todas as informações, clique em **Salvar** para concluir o cadastro do adiantamento.
