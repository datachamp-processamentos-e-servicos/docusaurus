# Retornos Bancários  Ⓧ

:::note[Antes de começar]

- Os arquivos de retorno são disponibilizados pelo banco após o processamento das cobranças.
- Eles pormitem atualizar a situação dos títulos no sistema conforme as ocorrências registradas pela instituição financeira.

## Como acessar

acesse:

**Financeiro → Cobrança Bancária → Retornos**

1. Utilize os filtros para localizar os arquivos desejados.
2. Clique sobre um registro para visualizar suas informações.
3. Utilize as opções **Retorno** ou **Retorno Via API** para importar novos arquivos quando necessário.

---

## Consulta dos Retornos

A listagem apresenta os arquivos de retorno já processados pelo sistema.

Entre as informações disponíveis estão:

- Data;
- Tipo do arquivo;
- Finalidade;
- Nome do arquivo;
- Portador bancário.

---

## Detalhes do Arquivo

Ao abrir um registro é possível consultar as informações gerais do arquivo, bem como visualizar seu conteúdo completo para conferência.

---

## Finalizando

Após a importação e processamento do retorno, os títulos relacionados poderão ser atualizados conforme as ocorrências informadas pelo banco.
