# Retornos Bancários  Ⓧ

## Visão Geral

A tela de Retornos Bancários é utilizada para consultar os arquivos de retorno enviados pelos bancos após o processamento das cobranças.

Por meio desta funcionalidade, é possível acompanhar registros bancários, localizar arquivos processados e gerar relatórios relacionados às cobranças.

---
## Observações Importantes

- Os retornos dependem do processamento realizado pelo banco.
- Utilize filtros de data para localizar arquivos com mais facilidade.
- Os relatórios podem ser exportados em PDF.
- Os arquivos XML podem ser baixados para consulta ou integração.

---

## Principais Funcionalidades

### Consulta de Retornos

Permite consultar os retornos recebidos dos bancos.

Utilize os filtros disponíveis para localizar rapidamente os arquivos desejados.

## Pesquisa Avançada

Clique em **Pesquisa Avançada** para localizar retornos específicos.

### Filtros Disponíveis

### Nome do Arquivo

Localiza retornos através do nome do arquivo.

#### Finalidade

Permite pesquisar retornos conforme sua finalidade.

As opções podem incluir:

- Cobrança
- Pagamento

#### Portador

Permite Localizar retornos através do banco portador.

#### Filtro de Data

Permite pesquisar arquivos dentro de um período específico.

## Mais Ações

### Relatórios

Gera relatórios relacionados às cobranças bancárias processadas.

### PDF

Exporta o relatório para formato PDF.

### Download

Realiza o download do relatório bancário.

### XML

Permite baixar o arquivo XML do Relatório bancário.

## Dúvidas Frequentes

### O que é um retorno bancário?

É o arquivo enviado pelo banco contendo informações sobre o processamento dos boletos registrados.

### Como localizar um retorno específico?

É o arquivo enviado pelo banco contendo informações sobre o processamento dos boletos registrados.

### Como localizar um retorno específico?

Utilizw a Pesquisa Avançada e informe o nome do arquivo, banco ou período desejado.

### Como gerar um relatório dos retornos?

Acesse **Mais Ações > Relatórios**.

### Como baixar um retorno bancário?

Acesse **Mais Ações > download**.

### Posso exportar o relatório para PDF?

Sim. Utilize a opção **PDF** disponível no menu de ações.
