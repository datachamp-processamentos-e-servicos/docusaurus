Gerar Remessa Bancária

Passo a Passo – Geração e Importação de Remessa Bancária no Sentus

1. Acessar o menu de Remessas​

No sistema Sentus, acesse:

Financeiro → Cobrança Bancária → Remessas

Nesta tela serão exibidas as remessas já geradas anteriormente.

2. Gerar uma nova remessa​

Clique no botão:

+ Remessa

Será aberta a tela:

Gerar Arquivo de Remessa Bancária

Preencha os campos:

  • Portador
  • Finalidade
  • Período
  • Data Inicial
  • Data Final

Após preencher os filtros, clique em:

Pesquisar

3. Selecionar os títulos​

O sistema exibirá a lista de títulos disponíveis para envio.

Verifique:

  • Participante
  • Emissão
  • Vencimento
  • Valor
  • Situação

Marque os títulos desejados.

Depois clique em:

Remessa

O sistema irá gerar o arquivo de remessa bancária.

4. Confirmar geração da remessa​

Após a geração, o arquivo ficará disponível na tela de Remessas.

Serão exibidas informações como:

  • Nome do arquivo
  • Data
  • Tipo do arquivo
  • Portador

O status aparecerá em verde indicando que a remessa foi gerada com sucesso.

5. Acessar o portal do banco​

Abra o portal do banco (exemplo mostrado no vídeo: Sicredi).

No menu do banco acesse:

Cobrança → Transferência de Arquivos → Importar Arquivos

6. Importar o arquivo de remessa​

Na tela de importação:

  1. Clique em Escolher arquivo.
  2. Localize o arquivo gerado pelo Sentus.
  3. Selecione o arquivo de remessa.
  4. Clique em Abrir.
  5. Clique em Incluir no portal do banco.

O banco fará a validação e processamento do arquivo.

7. Importar arquivo de retorno no Sentus​

Após o processamento pelo banco, faça o download do arquivo de retorno.

No Sentus acesse:

Financeiro → Cobrança Bancária → Retornos

Clique em:

+ Retorno

8. Fazer upload do retorno​

Na janela exibida:

  1. Clique em Escolher arquivo.
  2. Selecione o arquivo de retorno baixado do banco.
  3. Clique em Upload.

O sistema fará a leitura e atualização automática dos títulos.

Observações Importantes​

Sempre confira os títulos antes de gerar a remessa.

Certifique-se de utilizar o portador correto.

Não altere manualmente o conteúdo do arquivo gerado.

O retorno bancário é necessário para atualizar pagamentos, rejeições e baixas automaticamente no sistema.
