# Cheques Emitidos e Recebidos

## Visão Geral

A tela de Cheques é utilizada para controlar os cheques emitidos e recebidos pela empresa.

Por meio desta funcionalidade, é possível registrar informações financeiras, acompanhar vencimentos, controlar liquidações e consultar a situação dos cheques cadastrados.

---

## Observações Importantes

- Os campos marcados com (*) são obrigatórios.
- O cadastro pode ser utilizado tanto para cheques emitidos quanto para cheques recebidos.
- Algumas funcionalidades dependem das permissões do usuário.
- A liquidação em lote permite processar diversos cheques simultaneamente.
- Os filtros de pesquisa ajudam a localizar registros rapidamente.

---

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para cadastrar um cheque.

Preencha os campos obrigatórios e informe os dados financeiros relacionados ao cheque.

Após concluir o preenchimento, clique em **Salvar**.

### Informações do Cheque

Nesta seção são registradas aos dados principais do cheque.

Os principais campos disponíveis são:

- Emitente
- Cliente ou Fornecedor
- Portador
- Série
- Número
- Nominal
- Emissão
- Vencimento
- Valor
- Grupo Financeiro
- Operação Contábil
- Conta Contábil
- Situação
- Observações

A situação permite acompanhar o status do cheque, como em aberto ou liquidado.

### Informações do Pagamento

Permite registrar os dados relacionados à liquidação do cheque.

Os principais campos incluem:

- Código de Barras
- Data de Pagamento
- Juros
- Desconto

Essas informações auxiliam no controle financeiro e na conciliação dos pagamentos.

## Menu

### Contabilização

Permite realizar processo de contabilização dos cheques.

Esta funcionalidade é disponibilizada apenas para usuários com permissões administrativas.

### Liquidação em Lote

Permite liquidar vários cheques simultaneamente.

Durante a liquidação podem ser informados:

- Data de Pagamento
- Valor Total
- Descontos
- Juros
- Operação Contábil

Essa funcionalidade agiliza o processamento de grandes volumes de cheques.

### Totalizar

Exibe os valores totais dos cheques cadastrados e permite acompanhar os valores em aberto.

### Imprimir

Gera um relatório ou comprovante com as informações dos cheques cadastrados.

### Exportar 

Exporta os dados exibidos na tela para um arquivo Excel.

Após a geração, o download é realizado automaticamente.

## Dúvidas Frequentes

### Como cadastrar um cheque?

Clique em **+ Novo**, preencha os dados obrigatórios e clique em **Salvar**.

### Como localizar um cheque cadastrado?

Utilize a **Pesquisa Avançada** e informe os filtros desejados.

### Como liquidar vários cheques ao mesmo tempo?

Acesse **Menu > Liquidação em Lote**, informe os dados necessários e confirme a operação.

### Como verificar os valores em aberto?

Utilize a opção **Totalizar** disponível no menu da tela.

### Como exportar os registros?

Acesse **Menu > Exportar** para gerar uma planilha Excel com os dados exibidos.

### Quem pode acessar a contabilização?

A funcionalidade de contabilização está disponível apenas para usuários com permissões administrativas.
