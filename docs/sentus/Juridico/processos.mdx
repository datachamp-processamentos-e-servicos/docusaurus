## Cadastrar Novo Processo

Acesse:

**Jurídico → Processos**

1. Clique em **+ Novo** para cadastrar um processo ou selecione um existente para editá-lo.
2. Preencha os dados do processo.
3. Clique em **Avançar** para prosseguir para as próximas etapas.
4. Após concluir todas as etapas, clique em **Salvar**.

## Dados do Processo

Nesta primeira etapa são informadas as informações gerais do processo.

Os principais campos são:

- **Processo Nr.:** número do processo.
- **Emissão:** data do cadastro.
- **Valor:** valor total do processo
- **Situação:** situação atual do processo.
- **Reclamente:** participante reclamante.
- **Advogado:** advogado responsável.
- **Observação:** informações complementares sobre o processo.

## Itens

Nesta etapa são cadastradas os componentes financeiros do processo.

Os principais campos são:

- **Emissão:** data do componente.
- **Valor:** valor do componente.
- **Situação:** situação do lançamento.
- **Grupo:** grupo financeiro correspondente.
- **Centro de Custo:** centro de custo relacionado.
- **Observação:** informações adicionais.

Utilize os botões:

- **+ Novo** para iniciar um novo componente.
- **Confirmar** para adicioná-lo à lista.
- **Editar** e **Excluir** para realizar alterações nos componentes cadastrados.

Na parte inferior da tela é apresentado um resumo com os valores separados por categoria e o **Total** do processo.

## Financeiro

Após concluir os componentes, avance para a etapa **Financeiro**, onde são gerados os lançamentos financeiros do processo.

Os principais campos são:

- **Parcela:** identificação da parcela.
- **Emissão:** data do lançamento.
- **Vencimento:** data de vencimento.
- **Valor:** valor da parcela.
- **Grupo:** grupo financeiro utilizado.
- **Centro de Custo:** centro de custo do lançamento.
- **Observações/Hist. Contábil:** histórico que será utilizado no lançamento financeiro.

Utilize:

- **+ Novo** para criar uma parcela manualmente, quando necessário.
- **Confirmar** para adicionar a parcela.
- **Editar** para alterar um lançamento.
- **Abrir Pasta** para acessar os documentos vinculados ao lançamento.

Na parte inferior é exibida a relação das parcelas geradas, contendo informações de emissão, vencimento, saldo, grupo financeiro e o **Total** do processo.

**Finalizando**

Após revisar todas as informações do processo, Clique em **Salvar** para concluir o cadastro.
