# Cotações Ⓧ

## Visão Geral

A tela de Cotações é utilizada para solicitar, comparar e acompanhar propostas de fornecedores para aquisição de produtos ou serviços.

Por meio desta funcionalidade, é possível registrar cotações, vincular fornecedores, controlar preços informados e acompanhar o processo até a geração da compra.

---

## Observações Importantes

- Os campos marcados com (*) são obrigatórios.
- A Data Controle limita o período disponível para preenchimento da cotação.
- Revise os itens e fornecedores antes de concluir o cadastro.
- Utilize tags para facilitar futuras pesquisas.
- Mantenha as informações da cotação atualizadas para garantir comparações corretas entre fornecedores.

---

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para criar uma nova cotação.

Preencha os campos obrigatórios e clique em **Salvar** para concluir o cadastro.

### Editar

Localize a cotação desejada e selecione a opção de edição.

Caso exista campo de CPF/CNPJ no cadastro, não altere essa informação após a criação do registro, pois o sistema poderá considerar como um novo cadastro.

### Informações da Cotação

Os principais dados da cotação são:

#### Número

Identificação da tomada de preços utilizada para agrupar cotações relacionadas.

#### Emissão

Data e hora de criação da cotação.

#### Data Controle

Define a data limite para preenchimento da cotação pelo fornecedor.

Após esta data, não será mais possível realizar alterações ou informar preços.

#### Situação

Define o status atual da cotação.

Exemplos:

- Aguardando
- Aprovada
- Autorizada
- Concluída
- Encerrada
- Arquivada
- Cancelada
- Em Cotação
- Preenchida
- Não Preenchida

### Participantes

Permite vincular os fornecedores participantes da cotação.

A pesquisa pode ser realizada por:

- Nome
- Código
- CPF/CNPJ

### Itens da Cotação

Permite incluir os produtos ou serviços que serão cotados.

Os principais campos são:

- Referência
- Produto/Serviço
- Quantidade
- Valor Unitário
- Valor Total
- Desconto
- Acréscimos
- Custo

Também podem ser informadas informações tributárias quando necessário.

### Condições Comerciais

Permite definir informações complementares da negociação.

#### Forma de Pagamento

Define a condição de pagamento da cotação.

#### Forma de Pagamento

Define a condição de pagamento da cotação.

#### Modalidade de Frete

Define como será realizado o transporte da mercadoria.

Exemplos:

- CIF
- FOB
- FCA/FAS
- Próprio Remetente
- Próprio Destinatário
- Sem Frete

#### Tags

Permite categorizar ou identificar a cotação.

#### Observações

Utilizado para registrar informações complementares relacionadas à negociação.

## Dúvidas Frequentes

### Como criar uma nova cotação?

Clique em **+ Novo**, informe os dados da cotação, adicione os fornecedores e os itens desejados e clique em **Salvar**.

### Como adicionar fornecedores?

Selecione a cotação desejada e utilize **Mais Ações > Adicionar Fornecedor**.

### Como adicionar novos produtos?

Selecione a cotação e utilize **Mais Ações > Adicionar Item**.

### Como exportar uma cotação?

Acesse **Menu > Exportar** para gerar uma planilha com os dados da cotação.

### Como enviar uma cotação por e-mail?

Acesse **Menu > Enviar E-mail**, informe o destinatário e confirme o envio.

### Como localizar uma cotação específica?

Utilize a **Pesquisa Avançada** e filtre por número, participante, item, situação ou período.
