# Ordens de Compra Ⓧ

:::note[Antes de começar]

- O preenchimento dos campos com **(*)** é obrigatório.
- Ao posicionar o cursor do mouse sobre um campo, o sistema exibe uma breve explicação sobre o preenchimento.

Acesse:

**Comercial → Compras → Ordem de Compra**

1. Clique em **+ Novo** para criar uma nova ordem de compra.
2. Preencha as informações de cada etapa.
3. Clique em **Salvar** para concluir o cadastro.

## Dados da Ordem

Nesta etapa são informadas as informações gerais da ordem de compra.

Preencha os campos necessários, lembrando que os campos com **(*)** são obrigatórios.

**Dados da Ordem**

Informe:

- Número.
- Data de Emissão.
- Data de Entrega.
- Data Limite.
- Situação.

**Fornecedor**

Selecione o fornecedor através da pesquisa por:

- Nome.
- CNPJ/CPF.
- Código.

Após selecionar o fornecedor, seus dados cadastrais serão carregados automaticamente.

**Informações Gerais**

Configure, quando necessário:

- Portador.
- Forma de Pagamento.
- Grupo.
- Centro de Custo.
- Comprador.
- Emitente.

**Informações Adicionais**

Utilize este campo para registrar observações gerais da ordem de compra.

## Itens

Cadastre os produtos ou serviços que farão parte da ordem.

Informe:

- Referência.
- Produto.
- Operação.
- Quantidade.
- Valor Unitário.
- Valor da Mercadoria.
- Desconto.
- Informações Adicionais.

Quando necessário, informe também:

- Alíquota ICMS.
- Alíquota ICMS ST.
- DIFAL/ST/FCP.
- Alíquota IPI.
- Alíquota PIS.
- Alíquota COFINS.

Após preencher os dados do item, clique em **Confirmar** para adicioná-lo à lista.

## Transporte

Informe os dados de transporte da mercadoria.

Configure:

- Modalidade de Frete.
- Transportadora.
- Valor Total do Frete.

As modalidades disponíveis são:

- CIF – Por conta do emitente.
- FOB – Por conta do destinatário/remetente.
- FCA/FAS – Por conta de terceiros.
- Próprio por conta do Remetente.
- Próprio por conta do Destinatário.
- Sem Frete.

## Informações Adicionais

Quando necessário, informe:

- Observações da ordem de compra.
- Tags para identificação do documento.

**Finalizando**

Após revisar todas as informações e os itens cadastrados, clique em **Salvar** para concluir a Ordem de Compra.
