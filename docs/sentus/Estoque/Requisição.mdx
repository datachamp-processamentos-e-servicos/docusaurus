## Como cadastrar uma Requisição

**Acesse:**

**Estoque → Requisição**

1. Clique em **+ Novo** para criar uma requisição.
2. Preencha as informações da requisição.
3. Adicione os itens desejados.
4. Clique em **Confirmar** para incluir cada item na lista.
5. Após revisar as informações, clique em **Salvar**.

## Informações

Nesta tela são registradas as requisições de materiais para movimentação de estoque.

- Solicitante.
- Depósito.
- Setor.
- Centro de Custo.
- Operação Contábil.
- Máquina/Equipamento.
- Utilização.
- Observação Contábil.

## Itens da Requisição

Nesta seção são adicionados os itens que farão parte da requisição.

- Item.
- Quantidade.
- Estoque disponível.
- Lista de itens incluídos.

## Observações

- Utilize o botão **+ Novo** para adicionar itens à requisição.
- Revise os itens antes de salvar o cadastro.
- Após salvar, a requisição ficará disponível para consulta e processamento.
