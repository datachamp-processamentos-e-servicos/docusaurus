# Conferência Ⓧ

## Visão Geral

A tela de Conferência é utilizada para validar os saldos físicos dos produtos em estoque, permitindo comparar as quantidades existentes com os registros do sistema.

Esta funcionalidade auxilia no controle de divergências e na atualização das informações de estoque.

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para criar uma nova conferência.

Informe os dados necessários e clique em **Salvar** para concluir o cadastro.

### Informações da Conferência

Os principais campos disponíveis são:

- ID
- Data
- Tipo
- Origem
- Situação
- Referência
- Código de Barras (EAN)
- Item
- Depósito
- Quantidade
- Valor Unitário
- Custo Total
- Estoque Atual
- Operação
- Observação

### Conferência de Itens

Permite registrar as quantidades encontradas fisicamente durante a conferência do estoque.

## Menu

### Produção Sob Demanda

Permite consultar informações relacionadas à produção sob demanda.

### Zerar Saldo

Remove o saldo atual dos itens selecionados.

### Transferir Saldo

Permite transferir saldos entre registros.

### Imprimir

Gera uma versão para impressão da conferência.

### Exportar

Exporta os registros exibidos na tela.

## Ficha de Estoque

Exibe o histórico completo de movimentações do item selecionado.

## Dúvidas Frequentes

### Como criar uma conferência?

Clique em **+ Novo**, informe os dados necessários e salve o registro.

### Como localizar um item?

Utilize a Pesquisa Avançada e informe os filtros desejados.

### Como consultar o histórico de um produto?

Utilize a opção **Ficha de Estoque**.
