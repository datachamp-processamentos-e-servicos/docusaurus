## Consultar Conferência de Estoque

Acesse:

**Estoque → Conferência**

1. Informe os filtros desejados para localizar os itens;
2. Defina a data da posição do estoque e o depósito que será consultado;
3. Caso necessário, utilize filtros como referência, código de barras, nome, grupo, fornecedor ou situação;
4. Clique em **Pesquisar** para visualizar os resultados;
5. Para utilizar filtros adicionais, clique em **Pesquisa Avançada**.

## Pesquisa Avançada

A pesquisa avançada permite realizar consultas mais específicas quando os filtros principais não forem suficientes.

Entre os filtros disponíveis estão:

- NCM;
- Conta Contábil;
- Grupo Tributário e Subgrupo;
- CEST;
- Estoque;
- Marca;
- Tipo;
- Tags;
- cBenef;
- Ordenação dos resultados;
- Pesquisa por período.

Também é possível limitar a pesquisa ao emitente selecionado e utilizar informações adicionais para localizar produtos específicos.

Após configurar os filtros desejados, clique em **Pesquisar** para gerar a consulta.

## Informações da Consulta

A conferência apresenta a posição do estoque de acordo com os filtros informados.

Essa consulta é utilizada para verificar a disponibilidade dos produtos, localizar itens específicos e conferir informações antes da realização de movimentações, inventários ou demais operações relacionadas ao estoque.

Caso seja necessário iniciar uma nova pesquisa, utilize a opção **Limpar** para remover todos os filtros informados.

---

**Finalizando**

Após localizar os produtos desejados, utilize as informações apresentadas para realizar a conferência do estoque ou efetuar novas consultas conforme necessário.
