## Consultar Movimentações de Estoque

Acesse:

**Estoque → Mov. Estoque**

1. Informe o período que será consultado;
2. Selecione o depósito desejado;
3. Utilize os filtros disponíveis para localizar os produtos ou movimentações específicas;
4. Caso necessário, clique em **Pesquisa Avançada** para utilizar filtros adicionais;
5. Clique em **Pesquisar** para visualizar as movimentações encontradas.

## Informações da Consulta

A consulta de movimentações permite acompanhar todas as entradas e saídas de produtos realizadas no estoque durante um determinado período.

Essa ferramenta é utilizada para verificar o histórico de movimentações dos itens, identificar a origem das operações e facilitar auditorias e conferências do estoque.

Entre os principais filtros disponíveis estão:

- Período da movimentação;
- Depósito;
- Referência e nome do item;
- Grupo;
- Fornecedor;
- Situação do produto;
- Origem da entrada;
- Origem da saída.

## Pesquisa Avançada

Quando os filtros principais não forem suficientes, utilize a **Pesquisa Avançada** para realizar consultas mais detalhadas.

Nela é possível filtrar as movimentações por informações como:

- Código de barras;
- NCM;
- Conta Contábil;
- Grupo Tributário;
- Subgrupo;
- CEST;
- Marca;
- Tipo;
- Tags;
- cBenef;
- Ordenação dos resultados.

Esses filtros permitem localizar movimentações específicas de forma mais rápida e precisa.

---

**Finalizando**

Após realizar a consulta, utilize as informações apresentadas para acompanhar o histórico das movimentações, realizar conferências ou apoiar análises relacionadas ao controle de estoque.
