# Movimentação de Estoque

## Visão Geral

A tela de Movimentação de Estoque permite consultar todas as entradas, saídas e saldos dos produtos armazenados.

Por meio desta funcionalidade é possível acompanhar o histórico de movimentação dos itens e analisar o comportamento do estoque.

## Principais Funcionalidades

### Consulta de Movimentações

A tela apresenta informações como:

- Referência
- Nome
- Unidade
- Depósito
- Saldo Inicial
- Entradas
- Saídas
- Saldo Atual

### Análise de Estoque

Permite acompanhar a movimentação dos produtos dentro de um período específico.

## Menu

### Gerar Estoque

Atualiza as informações de movimentação e saldo.

### Imprimir

Gera uma versão para impressão dos registros.

### Exportar

Exporta os dados para planilha.

## Pesquisa Avançada

### Filtros Disponíveis

- Referência
- Código de Barras
- Nome
- Conta Contábil
- NCM
- Grupo Tributário
- CEST
- Grupo
- Subgrupo
- Tipo
- Estoque
- Marca
- Tags
- Situação
- Ordenação
- Observação

## Ficha de Estoque

Permite visualizar o histórico completo de movimentações do item.

## Dúvidas Frequentes

### Como consultar as movimentações de um produto?

Utilize a Pesquisa Avançada e informe o item desejado.

### Como visualizar entradas e saídas?

Os valores são exibidos diretamente na listagem da movimentação.

### Como exportar os dados?

Clique em **Menu > Exportar**.
