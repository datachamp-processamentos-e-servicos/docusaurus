## Consultar Visão Geral do Estoque

Acesse:

**Estoque → Visão Geral**

1. Informe a data da posição do estoque;
2. Selecione o depósito que será consultado;
3. Utilize os filtros disponíveis para localizar os produtos desejados;
4. Caso necessário, clique em **Pesquisa Avançada** para utilizar filtros adicionais;
5. Clique em **Pesquisar** para visualizar as informações do estoque.

## Informações da Consulta

A Visão Geral do Estoque permite consultar a posição dos produtos em uma determinada data, apresentando informações atualizadas conforme os filtros informados.

Essa consulta é utilizada para acompanhar a disponibilidade dos itens, verificar saldos e facilitar a análise do estoque antes da realização de movimentações, inventários ou compras.

Entre as principais opções de consulta estão:

- Data da posição do estoque;
- Depósito;
- Referência e nome do item;
- Grupo;
- Fornecedor;
- Situação do produto;
- Tipo de saldo apresentado.

## Pesquisa Avançada

A opção **Pesquisa Avançada** permite aplicar filtros adicionais para tornar a consulta mais específica.

Utilize essa opção sempre que for necessário localizar produtos com critérios mais detalhados, reduzindo a quantidade de resultados apresentados.

---

**Finalizando**

Após localizar os produtos desejados, utilize as informações apresentadas para acompanhar a posição do estoque e realizar as análises necessárias.
