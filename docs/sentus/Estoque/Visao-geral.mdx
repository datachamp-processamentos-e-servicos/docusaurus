# Visão Geral do Estoque Ⓧ

## Visão Geral

A tela de Visão Geral do Estoque permite consultar rapidamente os saldos dos produtos armazenados nos depósitos da empresa.

É uma ferramenta utilizada para consultas rápidas e acompanhamento da posição atual do estoque.

## Principais Funcionalidades

### Consulta de Estoque

A listagem apresenta:

- Referência
- Nome
- Unidade
- Quantidade
- Depósito

### Consulta de Saldo

Permite visualizar a quantidade disponível de cada item em estoque.

## Menu

### Imprimir

Gera uma versão para impressão da consulta.

### Exportar

Exporta os dados exibidos para planilha.

## Pesquisa Avançada

### Filtros Disponíveis

- Referência
- Código de Barras
- Nome
- Conta Contábil
- NCM
- Grupo Tributário
- CEST
- Grupo
- Subgrupo
- Tipo
- Estoque
- Marca
- Tags
- Situação
- Ordenação
- Observação

## Ficha de Estoque

Exibe:

- Saldo Inicial
- Entradas
- Saídas
- Saldo Final

## Relatórios

Permite gerar relatórios relacionados à posição e movimentação dos produtos em estoque.

## Dúvidas Frequentes

### Como consultar o saldo de um produto?

Utilize a Pesquisa Avançada e localize o item desejado.

### Como visualizar o histórico de movimentação?

Acesse a opção **Ficha de Estoque**.

### Como exportar os dados?

Clique em **Menu > Exportar**.
