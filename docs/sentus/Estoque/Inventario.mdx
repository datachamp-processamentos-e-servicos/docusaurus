## Como realizar um Inventário

**Acesse:**

**Estoque → Inventário**

1. Clique em **+ Novo** para iniciar um novo lançamento.
2. Selecione o **Item** que será inventariado.
3. Informe a **Quantidade** apurada na contagem física.
4. Verifique o **Depósito** e os valores calculados pelo sistema.
5. Caso necessário, preencha o campo **Observação**.
6. Após conferir todas as informações, clique em **Salvar**.

## Informações do Inventário

O inventário é utilizado para registrar a quantidade real dos itens em estoque, permitindo atualizar o saldo conforme a contagem física realizada.

Além da identificação do item, a tela apresenta informações de custo e valor, auxiliando na conferência antes da confirmação do lançamento.

Também é possível definir o depósito onde o ajuste será realizado e registrar observações para facilitar futuras consultas.

## Observações

- Realize o inventário após a conferência física dos itens.
- Certifique-se de selecionar o depósito correto antes de salvar.
- Utilize o campo de observação para registrar informações relevantes sobre o ajuste realizado.

**Finalizando**

Após revisar os dados do inventário, clique em **Salvar** para concluir o lançamento.
