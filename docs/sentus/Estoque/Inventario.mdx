# Inventário Ⓧ

## Visão Geral

A tela de Inventário é utilizada para registrar e controlar a posição dos produtos armazenados em estoque.

Por meio desta funcionalidade, é possível realizar fechamentos de inventário, consultar quantidades, recalcular valores e gerar informações para apuração fiscal e controle patrimonial.

---

## Observações Importantes

- Todos os campos marcados com (*) são obrigatórios.
- Revise as quantidades antes de realizar o fechamento.
- O fechamento do inventário impacta os controles de estoque.
- Utilize a opção Recalcular sempre que houver necessidade de atualização dos valores.
- Verifique o depósito correto antes de concluir o registro.

---

## Principais Funcionalidades

### Novo Cadastro

Clique em **+ Novo** para criar um novo inventário.

Preencha os campos obrigatórios e clique em **Salvar** para concluir o cadastro.

### Editar

Permite alterar as informações do inventário já cadastrado.

Caso exista campo de CPF/CNPJ no cadastro, não altere esta informação após a criação do registro, pois o sistema poderá interpretar a alteração como um novo cadastro.

### Informações do Inventário

#### ID

Identificação interna do inventário.

#### Data

Data de fechamento do inventário.

#### Item

Produto vinculado ao inventário.

#### Situação

Status atual do inventário.

#### Quantidade

Quantidade total do item inventariado.

#### Custo Total

Valor contábil total do inventário.

#### Valor Unitário

Valor unitário do item.

#### Preço de Venda

Maior preço de venda praticado nos últimos 12 meses.

#### Depósito

Depósito ao qual o inventário está vinculado.

#### Observação

Campo destinado ao registro de informações complementares. 

## Menu

### Fechamento

Permite realizar o fechamento do inventário.

Durante o fechamento é possível informar:

- Data Base
- Tipo de Fechamento
- Produção
- Teste

Após a confirmação, o sistema gera os dados consolidados do inventário. 

### Apuração ICMS

Gera uma relação dos produtos presentes no inventário para utilização em processos fiscais e apuração de ICMS. 

### Recalcular

Permite recalcular os valores do inventário com base nos parâmetros informados.

Esta opção é utilizada quando há necessidade de atualizar os valores registrados.

### Exportar

Exporta os dados exibidos na tela para uma planilha Excel.

## Exportar

Clique em **Menu > Exportar** para gerar uma planilha contendo os registros exibidos no inventário.

Após a geração, o download do arquivo será iniciado automaticamente. 

## Pesquisa Avançada

Utilize a Pesquisa Avançada para localizar inventários específicos.

### Filtros Disponíveis

- Somente Emitente Selecionado
- Item
- Depósito
- Tipo
- Observação

### Filtro por Data

Permite consultar registros por período utilizando a data de emissão. 

## Dúvidas Frequentes

### Como criar um novo inventário?

Clique em **+ Novo**, informe os dados necessários e clique em **Salvar**.

### Como realizar o fechamento do inventário?

Acesse **Menu > Fechamento**, informe os parâmetros desejados e confirme a operação.

### Para que serve a Apuração ICMS?

A funcionalidade gera uma relação dos produtos inventariados para utilização em controles fiscais.

### Quando utilizar a opção Recalcular?

Utilize quando houver necessidade de atualizar os valores registrados no inventário.

### Como exportar os dados?

Acesse **Menu > Exportar** para gerar uma planilha Excel.

### Como localizar um inventário específico?

Utilize a Pesquisa Avançada e informe filtros como item, depósito, observação ou período.
