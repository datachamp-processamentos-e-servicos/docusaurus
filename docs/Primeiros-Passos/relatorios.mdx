# RelatóriosⓍ

## Para gerar um relatório:

1. Acesse **Relatórios**;
2. Localize o relatório desejado;
3. Clique no ícone de **Configuração**;
4. Selecione o emitente;
5. Informe o período;
6. Preencha os filtros necessários;
7. Leia a seção **Observação**;
8. Clique em **Visualizar**;
9. Confira as informações apresentadas;
10. Utilize **PDF** ou **Outros Formatos** para exportar o resultado.
## Informações dos Relatórios

A tela apresenta todos os relatórios disponíveis para consulta e execução.

:::note[Antes de começar]

- Escolha o relatório pelo nome;
- Confira as datas;
- Leia a área **Observação**, quando ela aparecer.

:::

## Entendendo os Filtros

Os filtros ajudam o sistema a encontrar a informação certa.

Por exemplo:

- Para ver um cliente, preencha **Participante**;
- Para ver um vendedor, preencha **Representante**;
- Para ver um produto, preencha **Item**;
- Para ver tudo, deixe os campos extras vazios.

Quanto mais campos você preencher, menor será o resultado.

## Observação

Alguns relatórios mostram uma área chamada **Observação**.

Leia essa área antes de gerar o relatório, pois ela explica alguma regra importante.

## Exportar

Depois de gerar o relatório, você pode salvá-lo em:

- **PDF**;
- **Excel**;
- **CSV**;
- **TXT**.

Clique em **PDF** ou **Outros Formatos**, escolha uma opção e clique em **Exportar**.

## Nenhum Resultado Apareceu

Caso o relatório fique vazio:

1. Confira o Emitente;
2. Confira as datas;
3. Apague os filtros extras;
4. Clique novamente em **Visualizar**.

**Finalizando**

Escolha o relatório, informe as datas, clique em **Visualizar** e exporte o resultado quando precisar.
