# Relatórios

Acesse:

**Relatórios**

1. Localize o relatório desejado na lista ou utilize o campo de pesquisa para encontrá-lo.
2. Clique no ícone de **Configuração** correspondente ao relatório que deseja visualizar.
3. Preencha os parâmetros ou filtros disponíveis.
4. Clique em **Visualizar** para gerar o relatório.

## Informações dos Relatórios

A tela apresenta todos os relatórios disponíveis para consulta e execução.

As principais informações exibidas são:

- **Código:** identificação do relatório;
- **Título:** nome do relatório;
- **Módulo:** módulo ao qual o relatório pertence;
- **Orientação:** orientação padrão de impressão;
- **Ações:** permite configurar e executar o relatório.

Os filtros disponíveis variam conforme o relatório selecionado e podem incluir informações como:

- Situação;
- Grupo;
- Datas;
- Participantes;
- Entre outros.

O campo **Situação** permite selecionar um dos seguintes status:

- None (Nada/Nenhum);
- Ativo;
- Bloqueado;
- Inativo.

## Visualização e Exportação

Após gerar o relatório, as informações serão exibidas na tela.

Também é possível:

- Exportar o relatório em **PDF**;
- Exportar em **Outros Formatos** (PDF, Excel, CSV ou TXT);
- Definir o tipo de exportação, quando disponível:
  - **0 - Não se Aplica**;
  - **1 - Entre Linhas**;
  - **8 - Um PDF por Agrupamento**;
- Imprimir ou salvar o relatório conforme a necessidade.

## Finalizando

Após visualizar o relatório, utilize as opções de exportação ou impressão para salvar ou compartilhar as informações conforme a necessidade.
